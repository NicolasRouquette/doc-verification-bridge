    The axiom of unique choice. #

    We add in this file the axiom of unique choice, which is a weakening of the axiom of choice.

    Given a type Œ± such that there exists a unique element a : Œ±, the axiom of unique choice allows to select this element.

    The only version we will actually is the fact that, given a property P : Œ± ‚Üí Œ≤ ‚Üí Prop such that (h : ‚àÄ a, ‚àÉ! b, P a b), there is a function unique_choice_fun h : Œ± ‚Üí Œ≤ such that ‚àÄ a, P a (unique_choice_fun h a) (see unique_choice_fun and unique_choice_fun_spec). The construction of unique_choice_fun h can be done inconditionally in set theory, but in Lean's type theory we need an additional axiom.

    The axiom #

    • axiom_unique_choice: given a type Œ± such that h : ‚àÉ! (_ : Œ±), True, then axiom_unique_choice h gives the element.

    Main definitions #

    • unique_choice: given a property P : Œ± ‚Üí Prop such that h : ‚àÉ! a, P a, then unique_choice h gives the unique a : Œ± such that P a.
    • unique_choice_fun: given a property P : Œ± ‚Üí Œ≤ ‚Üí Prop such that for all a : Œ±, there exists a unique b : Œ≤ with P a b, then unique_choice_fun h gives a function Œ± ‚Üí Œ≤ selecting this unique b for each a.

    Main lemmas #

    • unique_choice_fun_spec: given a property P : Œ± ‚Üí Œ≤ ‚Üí Prop such that for all a : Œ±, there exists a unique b : Œ≤ with P a b, then for each a : Œ±, P a (unique_choice_fun h a) holds.
    • unique_choice_fun_unique: given a property P : Œ± ‚Üí Œ≤ ‚Üí Prop such that for all a : Œ±, there exists a unique b : Œ≤ with P a b, then for each a : Œ±, if b : Œ≤ satisfies P a b, then unique_choice_fun h a = b.
