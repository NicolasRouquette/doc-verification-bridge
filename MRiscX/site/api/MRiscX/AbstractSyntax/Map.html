                                                                                                k ≠ k' → (k ↦ v ; t).get k' = t.get k'

                                                                                                If a total map [t] is updated with some k ∈ α and v ∈ β (k ↦ v; t), and we search for some key [k'] with k ≠ k', we get the same result as when we would just search [t] for [k'].
