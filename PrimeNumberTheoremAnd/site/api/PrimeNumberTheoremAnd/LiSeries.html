    Series expansion for the logarithmic integral #

    We prove the well-known series expansion

    $$\mathrm{li}(x) = \gamma + \ln(\ln x) + \sum_{n=1}^{\infty} \frac{(\ln x)^n}{n \cdot n!}$$

    for $x > 1$, where $\gamma$ is the Euler–Mascheroni constant.

    Key theorems #

    • li_eq_eulerMascheroni_add_log_log_add_tsum : li x = eulerMascheroniConstant + log (log x) + ∑' n : ℕ, (log x) ^ (n + 1) / ((↑(n + 1) : ℝ) * ↑(n + 1).factorial)

    Strategy #

    This follows from the substitution $t = e^u$ in the definition $\mathrm{li}(x) = \mathrm{PV}\int_0^x \frac{dt}{\log t}$ which gives $\mathrm{li}(x) = \mathrm{PV}\int_{-\infty}^{\ln x} \frac{e^u}{u}\,du = \mathrm{Ei}(\ln x)$, and the classical series for the exponential integral $$\mathrm{Ei}(y) = \gamma + \ln y + \sum_{n=1}^{\infty} \frac{y^n}{n \cdot n!}$$ which is obtained by integrating $(e^u - 1)/u = \sum_{n=0}^{\infty} u^n/(n+1)!$ term by term on $[0, y]$.

    Part A: Power series for (eᵘ - 1)/u #

    Part B: Term-by-term integration #

    source
    theorem LiSeries.integral_exp_sub_one_div_eq_tsum (y : ℝ) :
    ∫ (u : ℝ) in 0..y, (Real.exp u - 1) / u = ∑' (n : ℕ), y ^ (n + 1) / (↑(n + 1) * ↑(n + 1).factorial)

    The integral ∫₀ʸ (eᵘ - 1)/u du = ∑ y^(n+1) / ((n+1) · (n+1)!).

    This is proved by integrating the power series (eᵘ-1)/u = ∑ uⁿ/(n+1)! term by term on [0, y]. Each term integrates to y^(n+1)/((n+1)(n+1)!).
