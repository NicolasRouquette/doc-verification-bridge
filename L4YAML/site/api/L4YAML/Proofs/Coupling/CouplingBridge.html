    Scanner ↔ Surface Syntax Bridge #

    Defines the formal correspondence between ScannerState (byte-level scanner over String) and SurfPos (character-level surface syntax over List Char), and proves coupling between scanner operations and surface syntax predicates.

    Architecture #

    The bridge works through CharsFromOffset, an inductive relation that connects byte offsets to character lists using the same raw String.Pos.Raw.get/next operations as the scanner.

    Sections #

    1. CharsFromOffset: byte offset → character list relation
    2. ScannerSurfCorr: scanner state ↔ surface position correspondence
    3. Peek/EOF: correspondence for scanner queries
    4. Advance: advance preserves correspondence
    5. Production coupling: scanner ops → surface predicates
    6. Composition helpers: building higher-level productions
