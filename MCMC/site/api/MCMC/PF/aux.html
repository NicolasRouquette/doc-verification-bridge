    Auxiliary lemmas for Perron-Frobenius and MCMC #

    Small results on the standard simplex, topology, matrices, and finsets shared by MCMC.PF. Prefer Mathlib statements when they already exist; this file keeps only what downstream modules use.

    Includes eq_mul_of_eq_div (solve a = c * b from c = a / b) and mul_div_mul_eq_div (cancel a common nonzero factor in a field fraction).
