    Absorbing chains and the fundamental matrix #

    This file contains the Chapter 4 absorbing-chain scaffold.

    The transient regime is encoded structurally through summability of the geometric series ∑ Q^k, so the Green-kernel objects expectedVisitsMatrix and expectedAbsorptionTime are only defined after providing a transient witness.
