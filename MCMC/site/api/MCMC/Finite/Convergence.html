      def MCMC.Finite.HasSpectralGap {n : Type u_1} [Fintype n] [DecidableEq n] (P : Matrix n n ℝ) :
      Prop

      Spectral gap encoded by: there exist π, r ∈ [0,1), and a positive block length k0, s.t. the entrywise error is bounded by r^(k / k0) for all k. This form still implies convergence, and is what we can prove from Dobrushin’s coefficient and primitivity (a positive power strictly contracts TV).
