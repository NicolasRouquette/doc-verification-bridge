        noncomputable def MCMC.Finite.mh_acceptance_prob {n : Type u_1} [Fintype n] (π : ↑(stdSimplex ℝ n)) (Q : Matrix n n ℝ) (x y : n) :
        ℝ

        The acceptance probability for a Metropolis-Hastings step. A(x, y) = min(1, (π(y) * Q(y, x)) / (π(x) * Q(x, y))).

        If the denominator is 0, the move is impossible to propose from x (if π(x)>0), or x is unreachable under π. We define the acceptance probability as 1 in this case.

          noncomputable def MCMC.Finite.metropolisHastingsKernel {n : Type u_1} [Fintype n] [DecidableEq n] (π : ↑(stdSimplex ℝ n)) (Q : Matrix n n ℝ) :
          Matrix n n ℝ

          The Metropolis-Hastings transition kernel (matrix). It is constructed by calculating the probability of proposal and acceptance (P_proposal), and then adding the remaining "rejection mass" (R) back to the diagonal to ensure stochasticity. P(x, y) = P_proposal(x, y) + δ(x, y) * R(x).
