      def MCMC.Finite.IsStationary {n : Type u_1} [Fintype n] (P : Matrix n n ℝ) (π : ↑(stdSimplex ℝ n)) :
      Prop

      A probability distribution π (represented as a column vector in the standard simplex) is stationary for P if Pᵀ *ᵥ π = π. (This corresponds to the row vector definition π_row P = π_row).
