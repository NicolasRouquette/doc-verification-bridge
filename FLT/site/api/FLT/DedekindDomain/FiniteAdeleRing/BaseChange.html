    Base change of adele rings. #

    If A is a Dedekind domain with field of fractions K, if L/K is a finite separable extension and if B is the integral closure of A in L, then B is also a Dedekind domain. Hence the rings of finite adeles ùî∏_K^‚àû and ùî∏_L^‚àû (defined using A and B) are defined.
