    structure CategoryTheory.Subfunctor {C : Type u} [Category.{v, u} C] (F : Functor C (Type w)) :
    Type (max u w)

    A Subfunctor of a functor consists of a subset of F.obj U for every U, compatible with the restriction maps F.map i.

    • obj (U : C) : Set (F.obj U)

      If G is a sub-functor of F, then the sections of G on U forms a subset of sections of F on U.

    • map {U V : C} (i : U ‚ü∂ V) : Set.MapsTo (F.map i) (self.obj U) (self.obj V)

      If G is a sub-functor of F and i : U ‚ü∂ V, then for each G-sections on U x, F i x is in F(V).
