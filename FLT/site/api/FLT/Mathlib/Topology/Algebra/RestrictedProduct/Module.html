    Restricted product of modules as a module over restricted product of rings #

    If R : Œπ ‚Üí Type* is a family of rings, B : (i : Œπ) ‚Üí Subring (R i) is a family of subrings, M : Œπ ‚Üí Type* is a family of types, with M i having an R i-module structure and C : (i : Œπ) ‚Üí Submodule (B i) (M i), then Œ† ≥ i, [M i, C i]_[ùìï] has a Œ† ≥ i, [R i, B i]_[ùìï]-module structure.

    Main definitions #

    • RestrictedProduct.module' is the module structure defined above.

    • RestrictedProduct.linearMap_component is the component M j ‚Üí‚Çó[R j] M' j given by a linear map Œ† ≥ i, [M i, C i]_[ùìï] ‚Üí‚Çó[Œ† ≥ i, [R i, B i]_[ùìï]] Œ† ≥ i, [M' i, C' i]_[ùìï].

    • ContinuousLinearEquiv.restrictedProductPi is the continuous linear equivalence between Œ† ≥ i, [(R i)^n, (B i)^n] and Œ† ≥ i, [(R i), (B i)]^n.

    Main theorems #

    • RestrictedProduct.isOpenMap_linearMap_of_surjective : a condition for a linear map to be surjective based on properties of its components.

    • Instances for ContinuousSMul (Œ† ≥ i, [R i, B i]_[ùìü T]) (Œ† ≥ i, [M i, C i]_[ùìü T]) and ContinuousSMul (Œ† ≥ i, [R i, B i]) (Œ† ≥ i, [M i, C i]).

    • RestrictedProduct.isModuleTopology : Œ† ≥ i, [M i, C i] has the Œ† ≥ i, [R i, B i]-module topology if (M i) have the (R i)-module topology and (B i) and (C i) are open.
