    Slices of space #

    i. Overview #

    In this module we will define the equivalence between Space d.succ and ℝ × Space d which extracts the ith coordinate on Space d.succ.

    ii. Key results #

    • slice : The continuous linear equivalence between Space d.succ and ℝ × Space d extracting the ith coordinate.
