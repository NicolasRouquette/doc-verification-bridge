    Charge spectrum which minimally allows terms #

    i. Overview #

    We can say that a charge spectrum x : ChargeSpectrum ğ“© minimally allows a potential term T : PotentialTerm if it allows the term T and no strict subset of x allows T.

    That is to say, you need all of the charges in x to allow the term T.

    We show that any charge spectrum which allows T has a subset which minimally allows T.

    We show that every charge spectrum which minimally allows T is of the form allowsTermForm a b c T for some a b c : ğ“©, and the reverse is true for T not equal to W1 or W2.

    ii. Key results #

    • MinimallyAllowsTerm : Predicate on charge spectra which is true if the charge spectrum minimally allows a potential term.
    • allowsTerm_iff_subset_minimallyAllowsTerm : A charge spectrum which allows a term has a subset which minimally allows the term, and vice versa.
    • eq_allowsTermForm_of_minimallyAllowsTerm : Any charge spectrum which minimally allows a term is of the form allowsTermForm a b c T for some a b c : ğ“©.
