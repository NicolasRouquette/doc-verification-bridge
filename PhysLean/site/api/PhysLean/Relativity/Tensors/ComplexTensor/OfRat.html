    Basis for tensors in a tensor species #

    source
    noncomputable def complexLorentzTensor.ofRat {n : ℕ} {c : Fin n → Color} :
    (TensorSpecies.Tensor.ComponentIdx c → PhysLean.RatComplexNum) →ₛₗ[PhysLean.RatComplexNum.toComplexNum] complexLorentzTensor.Tensor c

    A complex Lorentz tensor from a map (Π j, Fin (complexLorentzTensor.repDim (c j))) → RatComplexNum. All complex Lorentz tensors with rational coefficients with respect to the basis are of this form.
