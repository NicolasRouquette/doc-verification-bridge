    The Lorentz Algebra #

    We define

    • Define lorentzAlgebra via LieAlgebra.Orthogonal.so' as a subalgebra of Matrix (Fin 1 ⊕ Fin 3) (Fin 1 ⊕ Fin 3) ℝ.
    • In mem_iff prove that a matrix is in the Lorentz algebra if and only if it satisfies the condition Aᵀ * η = - η * A.
