        def Aesop.addLetDeclsToSimpTheorems (ctx : Lean.Meta.Simp.Context) :
        Lean.MetaM Lean.Meta.Simp.Context

        Add all let hypotheses in the local context as simp theorems.

        Background: by default, in the goal x : _ := v âŠ¢ P[x], simp does not substitute x by v in the target. The simp option zetaDelta can be used to make simp perform this substitution, but we don't want to set it because then Aesop simp would diverge from default simp, so we would have to adjust the aesop? output as well. Instead, we add let hypotheses explicitly. This way, simp? picks them up as well.

        See lean4#3520.
