    F.comp (gFunctor.comp (Limits.ker C)) ≅ F.comp (fFunctor.comp Abelian.im)

    The cokernel of the first map of an exact complex in an abelian category is naturally isomorphic to the coimage of the second map.

    Note that we use the extra functor F to avoid talking about the category of exact complex.
