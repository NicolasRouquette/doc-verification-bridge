    An approximation lemma #

    Let M be a filtered G-module, i.e. given G-submodules M_i ≤ M indexed by i : ℕ, such that M = lim[<-] M/M_i. Now fix q : ℕ and suppose that ∀ i : ℕ, H^(q+1)(G, M_i/M_{i+1}) = 0. Then H^(q+1)(G, M) = 0.
