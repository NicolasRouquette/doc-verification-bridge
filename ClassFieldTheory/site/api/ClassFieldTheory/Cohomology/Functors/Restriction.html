                            Given any short exact sewuence 0 → A → B → C → 0 in Rep R G and any subgroup H of G, the following diagram is commutative

                            Hⁿ(G,C) ⟶ H^{n+1}(G A) | | ↓ ↓ Hⁿ(H,C) ⟶ H^{n+1}(G A).

                            The vertical arrows are restriction and the horizontals are connecting homomorphisms.

                            For this, it would be sensible to define restriction as a natural transformation, so that it automatically commutes with the other maps. This requires us to first define cohomology as a functor.
