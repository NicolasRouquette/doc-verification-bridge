    Finite stochastic matrices and Markov chains #

    This file packages Chapter 4 style statements on finite stochastic matrices on top of the existing finite-state API from MCMC.Finite.Core and MCMC.Finite.Convergence.

    In particular, it avoids duplicating the existing definition MCMC.Finite.IsStationary: the row-vector convention is exposed through the equivalence isStationary_iff_vecMul.
