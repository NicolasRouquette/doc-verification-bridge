    theorem DenseRange.exists_matrix_det_ne_zero {X : Type u_1} {Y : Type u_2} {n : Type u_3} {R : Type u_4} [PseudoMetricSpace X] [Fintype n] [DecidableEq n] [SeminormedCommRing R] [NormOneClass R] [IsTopologicalRing R] {A : X → Matrix n n R} {f : Y → X} (hA : Continuous A) (hf : DenseRange f) {b : X} (hb : A b = 1) :
    ∃ (y : Y), (A (f y)).det ≠ 0

    Let A : X → Matrix n n R be a parameterisation of a matrix ring with entries in a seminormed commutative ring R by a metric space X and let f : Y → X be dense. If the identity matrix is in the parameterisation A, then there exists a y : Y such that A (f y) has non-zero determinant.
