    class IsProartinian (R : Type u_1) [CommRing R] [TopologicalSpace R] [IsTopologicalRing R] extends IsLinearTopology R R, T0Space R, CompleteSpace R :
    Prop

    A topological ring is proartinian if it is linearly topologized, complete hausdorff, and all its discrete quotients are artinian.

    This is also called the category of "pseudo-compact" rings in section 0 of Exp VII_B of SGA3.
