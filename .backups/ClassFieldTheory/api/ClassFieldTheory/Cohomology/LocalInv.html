    The local invariant #

    In this file, we define the carry cocycle, show that it is a two-cocycle and generates H²(ℤ/nℤ, ℤ) (with the trivial ℤ/nℤ-action on ℤ). In turn, this lets us define the local invariant as the isomorphism H²(ℤ/nℤ, ℤ) ≅ ℤ/nℤ.
