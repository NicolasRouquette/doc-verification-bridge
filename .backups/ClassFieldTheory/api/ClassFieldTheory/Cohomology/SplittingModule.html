                                            class Rep.split.FiniteClassFormation {R : Type} [CommRing R] {G : Type} [Group G] {M : Rep R G} (σ : ↑(groupCohomology.H2 M)) :
                                            Prop

                                            If M is an R[G]-module and σ : H²(G,M), we say σ is a finite class formation if for all subgroups H of G,

                                            1. H¹(H,M|H)=0;
                                            2. The R-module H²(H,M|H) is spanned as an R-module by Res(σ);
                                            3. The kernel of R → H²(H,M|H) is |H|R

                                            In other words, for all subgroups H of G, H¹(H,M)=0 and H²(H,M)=R/|H|R with the isomorphism given by sending 1 ∈ R/|H|R to σ.
