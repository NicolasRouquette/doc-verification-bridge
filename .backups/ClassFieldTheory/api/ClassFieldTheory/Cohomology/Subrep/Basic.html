    structure Subrep {k G : Type u} [CommRing k] [Monoid G] (A : Rep k G) extends Submodule k ↑A.V :
    Type u

    A subrepresentation is a submodule that is invariant under the G-action.

    • carrier : Set ↑A.V
    • add_mem' {a b : ↑A.V} : a ∈ self.carrier → b ∈ self.carrier → a + b ∈ self.carrier
    • zero_mem' : 0 ∈ self.carrier
    • smul_mem' (c : k) {x : ↑A.V} : x ∈ self.carrier → c • x ∈ self.carrier
    • le_comap (g : G) : self.toSubmodule ≤ Submodule.comap (A.ρ g) self.toSubmodule
