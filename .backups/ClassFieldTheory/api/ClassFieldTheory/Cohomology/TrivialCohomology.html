    Trivial (Tate) (co)homology #

    An object M : Rep R G is has trivial cohomology if Hⁿ(S, M)=0 for all n > 0 and all subgroup S of G.

    M has trivial homology if for all subgroups S and all n > 0 we have Hₙ(S, M) ≅ 0.

    M has trivial Tate cohomology if for all subgroups S and all n : ℤ we have Hⁿ_{Tate}(S, M) ≅ 0.

    We define these three classes of representation, and prove that they are preserved by isomorphisms.
