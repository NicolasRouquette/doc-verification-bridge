    Let R be a commutative ring and G a group. We define the augmentation module aug R G : Rep R G to be the kernel of the augmentation map "ε : R[G] ⟶ R".

    We construct the short exact sequence of H-modules for every subgroup H of G.

    0 ⟶ aug R G ⟶ R[G] ⟶ R ⟶ 0.

    In the case that G is finite, the representation R[G] is coinduced, and so has trivial cohomology (with respect to any subgroup H). This implies that the connecting homomorphisms give isomorphisms for all n > 0

    Hⁿ(H,R) ≅ Hⁿ⁺¹(H, aug R G).

    We also have isomorphisms

    H¹(H,aug R G) ≅ R ⧸ |H|R,

    H²(H, aug R G) ≅ 0, assuming IsAddTorsionFree R.
