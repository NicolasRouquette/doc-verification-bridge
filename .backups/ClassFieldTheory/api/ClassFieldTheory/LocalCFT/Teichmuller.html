    Teichmüller character #

    Given complete discrete valuation ring (R,m,k) with finite residue field k, we define Teichmüller character k →* R (a monoid homomorphism, i.e. preserves only multiplication).

    ... but we don't have CDVR so we will just do it for local fields.
