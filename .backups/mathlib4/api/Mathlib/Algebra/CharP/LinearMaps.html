    Characteristic of the ring of linear Maps #

    This file contains properties of the characteristic of the ring of linear maps. The characteristic of the ring of linear maps is determined by its base ring.

    Main Results #

    • Module.charP_end : For a commutative semiring R and an R-module M, the characteristic of R is equal to the characteristic of the R-linear endomorphisms of M when M contains a non-torsion element x.

    Notation #

    • R is a commutative semiring
    • M is an R-module
