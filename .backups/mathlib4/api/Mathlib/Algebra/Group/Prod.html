    Monoid, group etc. structures on M √ó N #

    In this file we define one-binop (Monoid, Group etc) structures on M √ó N. We also prove trivial simp lemmas, and define the following operations on MonoidHoms:

    • fst M N : M √ó N ‚Üí* M, snd M N : M √ó N ‚Üí* N: projections Prod.fst and Prod.snd as MonoidHoms;
    • inl M N : M ‚Üí* M √ó N, inr M N : N ‚Üí* M √ó N: inclusions of first/second monoid into the product;
    • f.prod g : M ‚Üí* N √ó P: sends x to (f x, g x);
    • When P is commutative, f.coprod g : M √ó N ‚Üí* P sends (x, y) to f x * g y (without the commutativity assumption on P, see MonoidHom.noncommPiCoprod);
    • f.prodMap g : M √ó N ‚Üí M' √ó N': Prod.map f g as a MonoidHom, sends (x, y) to (f x, g y).

    Main declarations #

    • mulMulHom/mulMonoidHom: Multiplication bundled as a multiplicative/monoid homomorphism.
    • divMonoidHom: Division bundled as a monoid homomorphism.
