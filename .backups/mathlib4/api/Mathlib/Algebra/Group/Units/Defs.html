    Units (i.e., invertible elements) of a monoid #

    An element of a Monoid is a unit if it has a two-sided inverse.

    Main declarations #

    • Units M: the group of units (i.e., invertible elements) of a monoid.
    • IsUnit x: a predicate asserting that x is a unit (i.e., invertible element) of a monoid.

    For both declarations, there is an additive counterpart: AddUnits and IsAddUnit. See also Prime, Associated, and Irreducible in Mathlib/Algebra/GroupWithZero/Associated.lean.

    Notation #

    We provide MË£ as notation for Units M, resembling the notation $R^{\times}$ for the units of a ring, which is common in mathematics.
