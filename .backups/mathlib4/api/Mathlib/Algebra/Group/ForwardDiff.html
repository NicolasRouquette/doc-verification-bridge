    Forward difference operators and Newton series #

    We define the forward difference operator, sending f to the function x ‚Ü¶ f (x + h) - f x for a given h (for any additive semigroup, taking values in an abelian group). The notation Œî_[h] is defined for this operator, scoped in namespace fwdDiff.

    We prove two key formulae about this operator:

    • shift_eq_sum_fwdDiff_iter: the Gregory-Newton formula, expressing f (y + n ‚Ä¢ h) as a linear combination of forward differences of f at y, for n ‚àà ‚Ñï;
    • fwdDiff_iter_eq_sum_shift: formula expressing the n-th forward difference of f at y as a linear combination of f (y + k ‚Ä¢ h) for 0 ‚â§ k ‚â§ n.

    We also prove some auxiliary results about iterated forward differences of the function n ‚Ü¶ n.choose k.

            Forward differences of polynomials #

            We prove formulae about the forward difference operator applied to polynomials:

            • fwdDiff_iter_pow_eq_zero_of_lt : The n-th forward difference of the function x ‚Ü¶ x^j is zero if j < n;
            • fwdDiff_iter_eq_factorial : The n-th forward difference of the function x ‚Ü¶ x^n is the constant function n!;
            • fwdDiff_iter_sum_mul_pow_eq_zero : The n-th forward difference of a polynomial of degree < n is zero (formulated using explicit sums over range n).
