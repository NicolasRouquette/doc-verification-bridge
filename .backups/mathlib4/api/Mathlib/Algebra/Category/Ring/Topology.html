    Topology on Hom(R, S) #

    In this file, we define topology on Hom(A, R) for a topological ring R, given by the coarsest topology that makes f ‚Ü¶ f x continuous for all x : A. Alternatively, given a presentation A = ‚Ñ§[x·µ¢]/I, this is the subspace topology Hom(A, R) ‚Ü™ Hom(‚Ñ§[x·µ¢], R) = R·∂•.

    Main results #

    • CommRingCat.HomTopology.isClosedEmbedding_precomp_of_surjective: Hom(A/I, R) is a closed subspace of Hom(A, R) if R is Hausdorff.

    • CommRingCat.HomTopology.isEmbedding_pushout: Hom(B ‚äó[A] C, R) has the subspace topology from Hom(B, R) √ó Hom(C, R).
