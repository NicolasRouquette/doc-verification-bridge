    Dual numbers #

    The dual numbers over R are of the form a + bÎµ, where a and b are typically elements of a commutative ring R, and Îµ is a symbol satisfying Îµ^2 = 0 that commutes with every other element. They are a special case of TrivSqZeroExt R M with M = R.

    Notation #

    In the DualNumber locale:

    • R[Îµ] is a shorthand for DualNumber R
    • Îµ is a shorthand for DualNumber.eps

    Implementation notes #

    Rather than duplicating the API of TrivSqZeroExt, this file reuses the functions there.
