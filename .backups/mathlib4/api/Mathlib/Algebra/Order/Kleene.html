    Kleene Algebras #

    This file defines idempotent semirings and Kleene algebras, which are used extensively in the theory of computation.

    An idempotent semiring is a semiring whose addition is idempotent. An idempotent semiring is naturally a semilattice by setting a â‰¤ b if a + b = b.

    A Kleene algebra is an idempotent semiring equipped with an additional unary operator âˆ—, the Kleene star.
