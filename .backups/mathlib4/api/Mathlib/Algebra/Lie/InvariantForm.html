    Lie algebras with non-degenerate invariant bilinear forms are semisimple #

    In this file we prove that a finite-dimensional Lie algebra over a field is semisimple if it does not have non-trivial abelian ideals and it admits a non-degenerate reflexive invariant bilinear form.
