    Trivial Lie modules and Abelian Lie algebras #

    The action of a Lie algebra L on a module M is trivial if ‚ÅÖx, m‚ÅÜ = 0 for all x ‚àà L and m ‚àà M. In the special case that M = L with the adjoint action, triviality corresponds to the concept of an Abelian Lie algebra.

    In this file we define these concepts and provide some related definitions and results.
