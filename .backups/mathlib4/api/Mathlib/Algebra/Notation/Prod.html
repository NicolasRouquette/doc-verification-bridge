    Arithmetic operators on (pairwise) product types #

    This file provides only the notation for (componentwise) 0, 1, +, *, ‚Ä¢, ^, ‚Åª¬π on (pairwise) product types. See Mathlib/Algebra/Group/Prod.lean for the Monoid and Group instances. There is also an instance of the Star notation typeclass, but no default notation is included.
