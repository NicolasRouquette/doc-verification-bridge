    Intrinsic star operation on linear maps #

    This file defines the star operation on linear maps: (star f) x = star (f (star x)). This corresponds to a map being star-preserving, i.e., a map is self-adjoint iff it is star-preserving.

    Implementation notes #

    Because there is a global star instance on H ‚Üí‚Çó[ùïú] H (defined as the linear map adjoint on finite-dimensional Hilbert spaces), which is mathematically distinct from this star, we provide this instance on WithConv (E ‚Üí‚Çó[R] F).

    The reason we chose WithConv is because together with the convolution product from Mathlib/RingTheory/Coalgebra/Convolution.lean, we get a ‚ãÜ-algebra when star (WithConv.toConv comul) = WithConv.toConv (comm ‚àò comul).
