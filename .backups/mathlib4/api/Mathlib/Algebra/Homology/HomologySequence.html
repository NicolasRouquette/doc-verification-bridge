    The homology sequence #

    If 0 ‚ü∂ X‚ÇÅ ‚ü∂ X‚ÇÇ ‚ü∂ X‚ÇÉ ‚ü∂ 0 is a short exact sequence in a category of complexes HomologicalComplex C c in an abelian category (i.e. S is a short complex in that category and satisfies hS : S.ShortExact), then whenever i and j are degrees such that hij : c.Rel i j, then there is a long exact sequence : ... ‚ü∂ S.X‚ÇÅ.homology i ‚ü∂ S.X‚ÇÇ.homology i ‚ü∂ S.X‚ÇÉ.homology i ‚ü∂ S.X‚ÇÅ.homology j ‚ü∂ ....     The proof is based on the snake lemma, similarly as it was originally done in the Liquid Tensor Experiment.
