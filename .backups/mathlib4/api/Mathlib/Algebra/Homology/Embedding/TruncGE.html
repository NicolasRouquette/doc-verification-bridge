    The canonical truncation #

    Given an embedding e : Embedding c c' of complex shapes which satisfies e.IsTruncGE and K : HomologicalComplex C c', we define K.truncGE' e : HomologicalComplex C c and K.truncGE e : HomologicalComplex C c' which are the canonical truncations of K relative to e.

    For example, if e is the embedding embeddingUpIntGE p of ComplexShape.up ‚Ñï in ComplexShape.up ‚Ñ§ which sends n : ‚Ñï to p + n and K : CochainComplex C ‚Ñ§, then K.truncGE' e : CochainComplex C ‚Ñï is the following complex:

    Q ‚ü∂ K.X (p + 1) ‚ü∂ K.X (p + 2) ‚ü∂ K.X (p + 3) ‚ü∂ ...

    where in degree 0, the object Q identifies to the cokernel of K.X (p - 1) ‚ü∂ K.X p (this is K.opcycles p). Then, the cochain complex K.truncGE e is indexed by ‚Ñ§, and has the following shape:

    ... ‚ü∂ 0 ‚ü∂ 0 ‚ü∂ 0 ‚ü∂ Q ‚ü∂ K.X (p + 1) ‚ü∂ K.X (p + 2) ‚ü∂ K.X (p + 3) ‚ü∂ ...

    where Q is in degree p.

    We also construct the canonical epimorphism K.œÄTruncGE e : K ‚ü∂ K.truncGE e.
