    The symmetry of the total complex of a bicomplex

    Let K : HomologicalComplex‚ÇÇ C c‚ÇÅ c‚ÇÇ be a bicomplex. If we assume both [TotalComplexShape c‚ÇÅ c‚ÇÇ c] and [TotalComplexShape c‚ÇÇ c‚ÇÅ c], we may form the total complex K.total c and K.flip.total c.

    In this file, we show that if we assume [TotalComplexShapeSymmetry c‚ÇÅ c‚ÇÇ c], then there is an isomorphism K.totalFlipIso c : K.flip.total c ‚âÖ K.total c.

    Moreover, if we also have [TotalComplexShapeSymmetry c‚ÇÇ c‚ÇÅ c] and that the signs are compatible [TotalComplexShapeSymmetrySymmetry c‚ÇÅ c‚ÇÇ c], then the isomorphisms K.totalFlipIso c and K.flip.totalFlipIso c are inverse to each other.
