    Shifting cochains

    Let C be a preadditive category. Given two cochain complexes (indexed by ‚Ñ§), the type of cochains HomComplex.Cochain K L n of degree n was introduced in Mathlib/Algebra/Homology/HomotopyCategory/HomComplex.lean. In this file, we study how these cochains behave with respect to the shift on the complexes K and L.

    When n, a, n' are integers such that h : n' + a = n, we obtain rightShiftAddEquiv K L n a n' h : Cochain K L n ‚âÉ+ Cochain K (L‚ü¶a‚üß) n'. This definition does not involve signs, but the analogous definition of leftShiftAddEquiv K L n a n' h' : Cochain K L n ‚âÉ+ Cochain (K‚ü¶a‚üß) L n' when h' : n + a = n' does involve signs, as we follow the conventions appearing in the introduction of [Brian Conrad's book Grothendieck duality and base change][conrad2000].
