    Combinations #

    Combinations in a type are finite subsets of given cardinality. This file provides some API for handling them in the context of a group action.

    • Set.powersetCard.subMulAction: When a group G acts on Œ±, the SubMulAction of G on powersetCard Œ± n.

    This induces a MulAction G (powersetCard Œ± n) instance. Then:

    • Set.powerSetCard.mulActionHom_of_embedding: the equivariant map from Fin n ‚Ü™ Œ± to powersetCard Œ± n.

    • Set.powersetCard.isPretransitive_of_isMultiplyPretransitive shows the pretransitivity of that action if the action of G on Œ± is n-pretransitive.

    • Set.powersetCard.isPretransitive shows that Equiv.Perm Œ± acts pretransitively on powersetCard Œ± n, for all n.

    • Set.powersetCard.compl: Given an equality m + n = Fintype.card Œ±, the complement of an n-combination, as an m-combination. This map is an equivariant map with respect to a group action on Œ±.

    • Set.powersetCard.mulActionHom_singleton: The obvious map from Œ± to powersetCard Œ± 1, as an equivariant map.
