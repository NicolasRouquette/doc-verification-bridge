    Limits in the category of elements #

    We show that if C has limits of shape I and A : C ‚•§ Type w preserves limits of shape I, then the category of elements of A has limits of shape I and the forgetful functor œÄ : A.Elements ‚•§ C creates them.

    Further results #

    • If A is (co)representable, then A.Elements has an initial object.
