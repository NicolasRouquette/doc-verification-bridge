    Filtered colimits commute with finite limits. #

    We show that for a functor F : J √ó K ‚•§ Type v, when J is finite and K is filtered, the universal morphism colimitLimitToLimitColimit F comparing the colimit (over K) of the limits (over J) with the limit of the colimits is an isomorphism.

    (In fact, to prove that it is injective only requires that J has finitely many objects.)

    References #

    • Borceux, Handbook of categorical algebra 1, Theorem 2.13.4
    • Stacks: Filtered colimits
