    Extradegeneracy for the Cech object #

    Let U : FormalCoproduct C. Let T be a terminal object in C. In this file, we construct an isomorphism from the Cech object U.cech that is defined for objects in FormalCoproduct to the general Cech nerve construction applied to the morphism from U to the terminal object. This isomorphism is used in order to show that, as an augmented object (over T), the Cech object U.cech has an extra degeneracy when there is a morphism T ‚ü∂ U.obj i‚ÇÄ for some i‚ÇÄ.
