    The Cech object for formal coproducts #

    Let C be a category that has finite products. In this file, we define a functor cechFunctor : FormalCoproduct C ‚•§ SimplicialObject (FormalCoproduct C) which sends a formal coproduct of objects U j (for j : Œπ) to the simplicial object which sends ‚¶ãn‚¶å to the formal coproduct, indexed by i : Fin (n + 1) ‚Üí Œπ, of the products of the objects U (i a) for all a : Fin (n + 1).
