    Formal Coproducts #

    In this file we construct the category of formal coproducts given a category.

    Main definitions #

    • FormalCoproduct: the category of formal coproducts, which are indexed sets of objects in a category. A morphism ‚àê i : X.I, X.obj i ‚ü∂ ‚àê j : Y.I, Y.obj j is given by a function f : X.I ‚Üí Y.I and maps X.obj i ‚ü∂ Y.obj (f i) for each i : X.I.
    • FormalCoproduct.eval : (C·µí·µñ ‚•§ A) ‚•§ ((FormalCoproduct C)·µí·µñ ‚•§ A): the universal property that a presheaf on C where the target category has arbitrary coproducts, can be extended to a presheaf on FormalCoproduct C.
