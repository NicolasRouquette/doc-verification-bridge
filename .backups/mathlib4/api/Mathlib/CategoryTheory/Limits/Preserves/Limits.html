    Isomorphisms about functors which preserve (co)limits #

    If G preserves limits, and C and D have limits, then for any diagram F : J ‚•§ C we have a canonical isomorphism preservesLimitsIso : G.obj (Limit F) ‚âÖ Limit (F ‚ãô G). We also show that we can commute IsLimit.lift of a preserved limit with Functor.mapCone: (PreservesLimit.preserves t).lift (G.mapCone c‚ÇÇ) = G.map (t.lift c‚ÇÇ).

    The duals of these are also given. For functors which preserve (co)limits of specific shapes, see the files in the directory Mathlib/CategoryTheory/Limits/Preserves/Shapes/.
