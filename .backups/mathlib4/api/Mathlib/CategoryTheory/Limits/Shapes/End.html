    Ends and coends #

    In this file, given a functor F : J·µí·µñ ‚•§ J ‚•§ C, we define its end end_ F, which is a suitable multiequalizer of the objects (F.obj (op j)).obj j for all j : J. For this shape of limits, cones are named wedges: the corresponding type is Wedge F.

    We also introduce coend F as multicoequalizers of (F.obj (op j)).obj j for all j : J. In these cases, cocones are named cowedges.

    • https://ncatlab.org/nlab/show/end
