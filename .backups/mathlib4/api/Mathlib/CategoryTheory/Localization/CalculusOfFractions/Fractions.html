    Lemmas on fractions #

    Let W : MorphismProperty C, and objects X and Y in C. In this file, we introduce structures like W.LeftFraction‚ÇÇ X Y which consists of two left fractions with the "same denominator" which shall be important in the construction of the preadditive structure on the localized category when C is preadditive and W has a left calculus of fractions.

    When W has a left calculus of fractions, we generalize the lemmas RightFraction.exists_leftFraction as RightFraction‚ÇÇ.exists_leftFraction‚ÇÇ, Localization.exists_leftFraction as Localization.exists_leftFraction‚ÇÇ and Localization.exists_leftFraction‚ÇÉ, and LeftFraction.map_eq_iff as LeftFraction‚ÇÇ.map_eq_iff.

    Implementation note #

    The lemmas in this file are phrased with data that is bundled into structures like LeftFraction‚ÇÇ or LeftFraction‚ÇÉ. It could have been possible to phrase them with "unbundled data". However, this would require introducing 4 or 5 variables instead of one. It is also very convenient to use dot notation. Many definitions have been made reducible so as to ease rewrites when this API is used.
