    Forgetful functors #

    A concrete category is a category C where the objects and morphisms correspond with types and (bundled) functions between these types, see the file Mathlib.CategoryTheory.ConcreteCategory.Basic

    Each concrete category C comes with a canonical faithful functor forget C : C ‚•§ Type*. We impose no restrictions on the category C, so Type has the identity forgetful functor.

    We say that a concrete category C admits a forgetful functor to a concrete category D, if it has a functor forget‚ÇÇ C D : C ‚•§ D such that (forget‚ÇÇ C D) ‚ãô (forget D) = forget C, see class HasForget‚ÇÇ. Due to Faithful.div_comp, it suffices to verify that forget‚ÇÇ.obj and forget‚ÇÇ.map agree with the equality above; then forget‚ÇÇ will satisfy the functor laws automatically, see HasForget‚ÇÇ.mk'.
