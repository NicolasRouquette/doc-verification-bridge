    Morphisms to a colimit in a Grothendieck abelian category #

    Let C : Type u be an abelian category [Category.{v} C] which satisfies IsGrothendieckAbelian.{w} C. We may expect that all the objects X : C are Œ∫-presentable for some regular cardinal Œ∫.
