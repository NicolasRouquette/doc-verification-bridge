    The Gabriel-Popescu theorem #

    We prove the following Gabriel-Popescu theorem: if C is a Grothendieck abelian category and G is a separator, then the functor preadditiveCoyonedaObj G : C ‚•§ ModuleCat (End G)·µê·µí·µñ sending X to Hom(G, X) is fully faithful and has an exact left adjoint.

    We closely follow the elementary proof given by Barry Mitchell.

    Future work #

    The left adjoint tensorObj G actually exists as soon as C is cocomplete and additive, so the construction could be generalized.

    The theorem as stated here implies that C is a Serre quotient of ModuleCat (End G)·µê·µí·µñ.
