    The image of a subfunctor #

    Given a morphism of type-valued functors p : F' ‚ü∂ F, we define its range Subfunctor.range p. More generally, if G' : Subfunctor F', we define G'.image p : Subfunctor F as the image of G' by f, and if G : Subfunctor F, we define its preimage G.preimage f : Subfunctor F'.
