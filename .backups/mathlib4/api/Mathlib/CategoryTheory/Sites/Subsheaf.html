    Subsheaf of types #

    We define the subsheaf of a type-valued presheaf.

    Main results #

    • CategoryTheory.Subfunctor.sheafify : The sheafification of a subpresheaf as a subpresheaf. Note that this is a sheaf only when the whole sheaf is.
    • CategoryTheory.Subfunctor.sheafify_isSheaf : The sheafification is a sheaf
    • CategoryTheory.Subfunctor.sheafifyLift : The descent of a map into a sheaf to the sheafification.
    • CategoryTheory.GrothendieckTopology.imageSheaf : The image sheaf of a morphism.
    • CategoryTheory.GrothendieckTopology.imageFactorization : The image sheaf as a Limits.imageFactorization.
