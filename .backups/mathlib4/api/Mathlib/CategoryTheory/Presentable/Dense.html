    Œ∫-presentable objects form a dense subcategory #

    In a Œ∫-accessible category C, the inclusion of the full subcategory of Œ∫-presentable objects is a dense functor. This expresses canonically any object X : C as a colimit of Œ∫-presentable objects, and we show that this is a Œ∫-filtered colimit.
