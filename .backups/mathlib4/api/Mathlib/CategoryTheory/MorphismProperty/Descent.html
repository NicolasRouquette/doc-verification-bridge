    Descent of morphism properties #

    Given morphism properties P and Q we say that P descends along Q (P.DescendsAlong Q), if whenever Q holds for X ‚ü∂ Z, P holds for X √ó[Z] Y ‚ü∂ X implies P holds for Y ‚ü∂ Z. Dually, we define P.CodescendsAlong Q.
