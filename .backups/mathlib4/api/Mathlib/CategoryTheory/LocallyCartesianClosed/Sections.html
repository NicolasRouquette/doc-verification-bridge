    The section functor as a right adjoint to the toOver functor #

    We show that in a cartesian monoidal category C, for any exponentiable object I, the functor toOver I : C ‚•§ Over I mapping an object X to the projection snd : X ‚äó I ‚ü∂ I in Over I has a right adjoint sections I : Over I ‚•§ C whose object part is the object of sections of X over I.

    In particular, if C is cartesian closed, then for all objects I in C, toOver I : C ‚•§ Over I has a right adjoint.
