    Subterminal objects #

    Subterminal objects are the objects which can be thought of as subobjects of the terminal object. In fact, the definition can be constructed to not require a terminal object, by defining A to be subterminal iff for any Z, there is at most one morphism Z ‚ü∂ A. An alternate definition is that the diagonal morphism A ‚ü∂ A ‚®Ø A is an isomorphism. In this file we define subterminal objects and show the equivalence of these three definitions.

    We also construct the subcategory of subterminal objects.
