    Semiconjugate and commuting maps #

    We define the following predicates:

    • Function.Semiconj: f : Œ± ‚Üí Œ≤ semiconjugates ga : Œ± ‚Üí Œ± to gb : Œ≤ ‚Üí Œ≤ if f ‚àò ga = gb ‚àò f;
    • Function.Semiconj‚ÇÇ: f : Œ± ‚Üí Œ≤ semiconjugates a binary operation ga : Œ± ‚Üí Œ± ‚Üí Œ± to gb : Œ≤ ‚Üí Œ≤ ‚Üí Œ≤ if f (ga x y) = gb (f x) (f y);
    • Function.Commute: f : Œ± ‚Üí Œ± commutes with g : Œ± ‚Üí Œ± if f ‚àò g = g ‚àò f, or equivalently Semiconj f g g.

    We say that f : Œ± ‚Üí Œ≤ semiconjugates ga : Œ± ‚Üí Œ± to gb : Œ≤ ‚Üí Œ≤ if f ‚àò ga = gb ‚àò f. We use ‚àÄ x, f (ga x) = gb (f x) as the definition, so given h : Function.Semiconj f ga gb and a : Œ±, we have h a : f (ga a) = gb (f a) and h.comp_eq : f ‚àò ga = gb ‚àò f.

      def Function.Commute {Œ± : Type u_1} (f g : Œ± ‚Üí Œ±) :
