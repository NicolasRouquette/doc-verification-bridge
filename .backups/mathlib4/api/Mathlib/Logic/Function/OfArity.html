    This provides Function.OfArity, such that OfArity Œ± Œ≤ 2 = Œ± ‚Üí Œ± ‚Üí Œ≤. Note that it is often preferable to use (Fin n ‚Üí Œ±) ‚Üí Œ≤ in place of OfArity n Œ± Œ≤.
