    A "module topology" for modules over a topological ring #

    If R is a topological ring acting on an additive abelian group A, we define the module topology to be the finest topology on A making both the maps ‚Ä¢ : R √ó A ‚Üí A and + : A √ó A ‚Üí A continuous (with all the products having the product topology). Note that - : A ‚Üí A is also automatically continuous as it is a ‚Ü¶ (-1) ‚Ä¢ a.

    This topology was suggested by Will Sawin here.

    Mathematical details #

    I (buzzard) don't know of any reference for this other than Sawin's mathoverflow answer, so I expand some of the details here.

    First note that the definition makes sense in far more generality (for example R just needs to be a topological space acting on an additive monoid).

    Next note that there is a finest topology with this property! Indeed, topologies on a fixed type form a complete lattice (infinite infs and sups exist). So if œÑ is the Inf of all the topologies on A which make + and ‚Ä¢ continuous, then the claim is that + and ‚Ä¢ are still continuous for œÑ (note that topologies are ordered so that finer topologies are smaller). To show + : A √ó A ‚Üí A is continuous we equivalently need to show that the pushforward of the product topology œÑ √ó œÑ along + is ‚â§ œÑ, and because œÑ is the greatest lower bound of the topologies making ‚Ä¢ and + continuous, it suffices to show that it's ‚â§ œÉ for any topology œÉ on A which makes + and ‚Ä¢ continuous. However pushforward and products are monotone, so œÑ √ó œÑ ‚â§ œÉ √ó œÉ, and the pushforward of œÉ √ó œÉ is ‚â§ œÉ because that's precisely the statement that + is continuous for œÉ. The proof for ‚Ä¢ follows mutatis mutandis.

    A topological module for a topological ring R is an R-module A with a topology making + and ‚Ä¢ continuous. The discussion so far has shown that the module topology makes an R-module A into a topological module, and moreover is the finest topology with this property.

    A crucial observation is that if M is a topological R-module, if A is an R-module with no topology, and if œÜ : A ‚Üí M is linear, then the pullback of M's topology to A is a topology making A into a topological module. Let's for example check that ‚Ä¢ is continuous. If U ‚äÜ A is open then by definition of the pullback topology, U = œÜ‚Åª¬π(V) for some open V ‚äÜ M, and now the pullback of U under ‚Ä¢ is just the pullback along the continuous map id √ó œÜ : R √ó A ‚Üí R √ó M of the preimage of V under the continuous map ‚Ä¢ : R √ó M ‚Üí M, so it's open. The proof for + is similar.

    As a consequence of this, we see that if œÜ : A ‚Üí M is a linear map between topological R-modules modules and if A has the module topology, then œÜ is automatically continuous. Indeed the argument above shows that if A ‚Üí M is linear then the module topology on A is ‚â§ the pullback of the module topology on M (because it's the inf of a set containing this topology) which is the definition of continuity.

    We also deduce that the module topology is a functor from the category of R-modules (R a topological ring) to the category of topological R-modules, and it is perhaps unsurprising that this is an adjoint to the forgetful functor. Indeed, if A is an R-module and M is a topological R-module, then the previous paragraph shows that the linear maps A ‚Üí M are precisely the continuous linear maps from (A with its module topology) to M, so the module topology is a left adjoint to the forgetful functor.

    This file develops the theory of the module topology.

    Main theorems #

    • IsTopologicalSemiring.toIsModuleTopology : IsModuleTopology R R. The module topology on R is R's topology.
    • IsModuleTopology.iso [IsModuleTopology R A] (e : A ‚âÉL[R] B) : IsModuleTopology R B. If A and B are R-modules with topologies, if e is a topological isomorphism between them, and if A has the module topology, then B has the module topology.
    • IsModuleTopology.instProd : If M and N are R-modules each equipped with the module topology, then the product topology on M √ó N is the module topology.
    • IsModuleTopology.instPi : Given a finite collection of R-modules each of which has the module topology, the product topology on the product module is the module topology.
    • IsModuleTopology.isTopologicalRing : If D is an R-algebra equipped with the module topology, and D is finite as an R-module, then D is a topological ring (that is, addition, negation and multiplication are continuous).

    Now say œÜ : A ‚Üí‚Çó[R] B is an R-linear map between R-modules equipped with the module topology.

    • IsModuleTopology.continuous_of_linearMap œÜ is the proof that œÜ is automatically continuous.
    • IsModuleTopology.isQuotientMap_of_surjective (hœÜ : Function.Surjective œÜ) is the proof that if furthermore œÜ is surjective then it is a quotient map, that is, the module topology on B is the pushforward of the module topology on A.

    Now say œà : A ‚Üí‚Çó[R] B ‚Üí‚Çó[R] C is an R-bilinear map between R-modules equipped with the module topology.

    The module topology, for a module A over a topological ring R. It's the finest topology making addition and the R-action continuous, or equivalently the finest topology making A into a topological R-module. More precisely it's the Inf of the set of topologies with these properties; theorems continuousSMul and continuousAdd show that the module topology also has these properties.
