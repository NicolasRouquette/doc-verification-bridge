    Closed sets #

    We define a few types of closed sets in a topological space.

    Main Definitions #

    For a topological space Œ±,

    • TopologicalSpace.Closeds Œ±: The type of closed sets.
    • TopologicalSpace.Clopens Œ±: The type of clopen sets.
