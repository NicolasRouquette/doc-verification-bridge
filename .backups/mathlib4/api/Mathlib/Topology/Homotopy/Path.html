    Homotopy between paths #

    In this file, we define a Homotopy between two Paths. In addition, we define a relation Homotopic on Paths, and prove that it is an equivalence relation.

    Definitions #

    • Path.Homotopy p‚ÇÄ p‚ÇÅ is the type of homotopies between paths p‚ÇÄ and p‚ÇÅ
    • Path.Homotopy.refl p is the constant homotopy between p and itself
    • Path.Homotopy.symm F is the Path.Homotopy p‚ÇÅ p‚ÇÄ defined by reversing the homotopy
    • Path.Homotopy.trans F G, where F : Path.Homotopy p‚ÇÄ p‚ÇÅ, G : Path.Homotopy p‚ÇÅ p‚ÇÇ is the Path.Homotopy p‚ÇÄ p‚ÇÇ defined by putting the first homotopy on [0, 1/2] and the second on [1/2, 1]
    • Path.Homotopy.hcomp F G, where F : Path.Homotopy p‚ÇÄ q‚ÇÄ and G : Path.Homotopy p‚ÇÅ q‚ÇÅ is a Path.Homotopy (p‚ÇÄ.trans p‚ÇÅ) (q‚ÇÄ.trans q‚ÇÅ)
    • Path.Homotopic p‚ÇÄ p‚ÇÅ is the relation saying that there is a homotopy between p‚ÇÄ and p‚ÇÅ
    • Path.Homotopic.setoid x‚ÇÄ x‚ÇÅ is the setoid on Paths from Path.Homotopic
