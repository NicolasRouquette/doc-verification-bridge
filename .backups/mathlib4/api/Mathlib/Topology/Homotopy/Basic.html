    Homotopy between functions #

    In this file, we define a homotopy between two functions f‚ÇÄ and f‚ÇÅ. First we define ContinuousMap.Homotopy between the two functions, with no restrictions on the intermediate maps. Then, as in the formalisation in HOL-Analysis, we define ContinuousMap.HomotopyWith f‚ÇÄ f‚ÇÅ P, for homotopies between f‚ÇÄ and f‚ÇÅ, where the intermediate maps satisfy the predicate P. Finally, we define ContinuousMap.HomotopyRel f‚ÇÄ f‚ÇÅ S, for homotopies between f‚ÇÄ and f‚ÇÅ which are fixed on S.

    Definitions #

    • ContinuousMap.Homotopy f‚ÇÄ f‚ÇÅ is the type of homotopies between f‚ÇÄ and f‚ÇÅ.
    • ContinuousMap.HomotopyWith f‚ÇÄ f‚ÇÅ P is the type of homotopies between f‚ÇÄ and f‚ÇÅ, where the intermediate maps satisfy the predicate P.
    • ContinuousMap.HomotopyRel f‚ÇÄ f‚ÇÅ S is the type of homotopies between f‚ÇÄ and f‚ÇÅ which are fixed on S.

    For each of the above, we have

    • refl f, which is the constant homotopy from f to f.
    • symm F, which reverses the homotopy F. For example, if F : ContinuousMap.Homotopy f‚ÇÄ f‚ÇÅ, then F.symm : ContinuousMap.Homotopy f‚ÇÅ f‚ÇÄ.
    • trans F G, which concatenates the homotopies F and G. For example, if F : ContinuousMap.Homotopy f‚ÇÄ f‚ÇÅ and G : ContinuousMap.Homotopy f‚ÇÅ f‚ÇÇ, then F.trans G : ContinuousMap.Homotopy f‚ÇÄ f‚ÇÇ.

    We also define the relations

    • ContinuousMap.Homotopic f‚ÇÄ f‚ÇÅ is defined to be Nonempty (ContinuousMap.Homotopy f‚ÇÄ f‚ÇÅ)
    • ContinuousMap.HomotopicWith f‚ÇÄ f‚ÇÅ P is defined to be Nonempty (ContinuousMap.HomotopyWith f‚ÇÄ f‚ÇÅ P)
    • ContinuousMap.HomotopicRel f‚ÇÄ f‚ÇÅ P is defined to be Nonempty (ContinuousMap.HomotopyRel f‚ÇÄ f‚ÇÅ P)

    and for ContinuousMap.homotopic and ContinuousMap.homotopic_rel, we also define the setoid and quotient in C(X, Y) by these relations.
