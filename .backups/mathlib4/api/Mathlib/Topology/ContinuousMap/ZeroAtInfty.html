    Continuous functions vanishing at infinity #

    The type of continuous functions vanishing at infinity. When the domain is compact C(Œ±, Œ≤) ‚âÉ C‚ÇÄ(Œ±, Œ≤) via the identity map. When the codomain is a metric space, every continuous map which vanishes at infinity is a bounded continuous function. When the domain is a locally compact space, this type has nice properties.
