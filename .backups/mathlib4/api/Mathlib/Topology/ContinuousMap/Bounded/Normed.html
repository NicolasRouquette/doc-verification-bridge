    Inheritance of normed algebraic structures by bounded continuous functions #

    For various types of normed algebraic structures Œ≤, we show in this file that the space of bounded continuous functions from Œ± to Œ≤ inherits the same normed structure, by using pointwise operations and checking that they are compatible with the uniform distance.
