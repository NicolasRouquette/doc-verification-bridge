    theorem isProperMap_iff_universally_closed {X : Type u} {Y : Type v} [TopologicalSpace X] [TopologicalSpace Y] {f : X ‚Üí Y} :
    IsProperMap f ‚Üî Continuous f ‚àß ‚àÄ (Z : Type u) [inst : TopologicalSpace Z], IsClosedMap (Prod.map f id)

    A map f : X ‚Üí Y is proper if and only if it is continuous and universally closed, in the sense that for any topological space Z, the map Prod.map f id : X √ó Z ‚Üí Y √ó Z is closed. Note that Z lives in the same universe as X here, but IsProperMap.universally_closed does not have this restriction.

    This is taken as the definition of properness in [N. Bourbaki, General Topology][bourbaki1966].
