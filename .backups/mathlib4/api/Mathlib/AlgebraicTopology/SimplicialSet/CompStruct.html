    Edges and "triangles" in simplicial sets #

    Given a simplicial set X, we introduce two types:

    • Given 0-simplices x‚ÇÄ and x‚ÇÅ, we define Edge x‚ÇÄ x‚ÇÅ which is the type of 1-simplices with faces x‚ÇÅ and x‚ÇÄ respectively;
    • Given 0-simplices x‚ÇÄ, x‚ÇÅ, x‚ÇÇ, edges e‚ÇÄ‚ÇÅ : Edge x‚ÇÄ x‚ÇÅ, e‚ÇÅ‚ÇÇ : Edge x‚ÇÅ x‚ÇÇ, e‚ÇÄ‚ÇÇ : Edge x‚ÇÄ x‚ÇÇ, a structure CompStruct e‚ÇÄ‚ÇÅ e‚ÇÅ‚ÇÇ e‚ÇÄ‚ÇÇ which records the data of a 2-simplex with faces e‚ÇÅ‚ÇÇ, e‚ÇÄ‚ÇÇ and e‚ÇÄ‚ÇÅ respectively. This data will allow to obtain relations in the homotopy category of X.

    (This API parallels similar definitions for 2-truncated simplicial sets. The definitions in this file are definitionally equal to their 2-truncated counterparts.)
