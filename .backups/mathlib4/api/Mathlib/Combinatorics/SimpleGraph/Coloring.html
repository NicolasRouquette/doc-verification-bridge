    Graph Coloring #

    This module defines colorings of simple graphs (also known as proper colorings in the literature). A graph coloring is the attribution of "colors" to all of its vertices such that adjacent vertices have different colors. A coloring can be represented as a homomorphism into a complete graph, whose vertices represent the colors.

    Main definitions #

    • G.Coloring Œ± is the type of Œ±-colorings of a simple graph G, with Œ± being the set of available colors. The type is defined to be homomorphisms from G into the complete graph on Œ±, and colorings have a coercion to V ‚Üí Œ±.

    • G.Colorable n is the proposition that G is n-colorable, which is whether there exists a coloring with at most n colors.

    • G.chromaticNumber is the minimal n such that G is n-colorable, or ‚ä§ if it cannot be colored with finitely many colors. (Cardinal-valued chromatic numbers are more niche, so we stick to ‚Ñï‚àû.) We write G.chromaticNumber ‚â† ‚ä§ to mean a graph is colorable with finitely many colors.

    • C.colorClass c is the set of vertices colored by c : Œ± in the coloring C : G.Coloring Œ±.

    • C.colorClasses is the set containing all color classes.
