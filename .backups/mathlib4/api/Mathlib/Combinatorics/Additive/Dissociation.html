    Dissociation and span #

    This file defines dissociation and span of sets in groups. These are analogs to the usual linear independence and linear span of sets in a vector space but where the scalars are only allowed to be 0 or Â±1. In characteristic 2 or 3, the two pairs of concepts are actually equivalent.

    Main declarations #

    • MulDissociated/AddDissociated: Predicate for a set to be dissociated.
    • Finset.mulSpan/Finset.addSpan: Span of a finset.
