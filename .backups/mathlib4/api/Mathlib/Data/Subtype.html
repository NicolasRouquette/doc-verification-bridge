    Subtypes #

    This file provides basic API for subtypes, which are defined in core.

    A subtype is a type made from restricting another type, say Œ±, to its elements that satisfy some predicate, say p : Œ± ‚Üí Prop. Specifically, it is the type of pairs ‚ü®val, property‚ü© where val : Œ± and property : p val. It is denoted Subtype p and notation {val : Œ± // p val} is available.

    A subtype has a natural coercion to the parent type, by coercing ‚ü®val, property‚ü© to val. As such, subtypes can be thought of as bundled sets, the difference being that elements of a set are still of type Œ± while elements of a subtype aren't.
