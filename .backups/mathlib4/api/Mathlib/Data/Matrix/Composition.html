    Composition of matrices #

    This file shows that M‚Çô(M‚Çò(R)) ‚âÉ M‚Çô‚Çò(R), M‚Çô(R·µí·µñ) ‚âÉ‚Çê[K] M‚Çô(R)·µí·µñ and also different levels of equivalence when R is an AddCommMonoid, Semiring, and Algebra over a CommSemiring K.

    Main results #

    • Matrix.comp is an equivalence between Matrix I J (Matrix K L R) and I √ó K by J √ó L matrices.
    • Matrix.swap is an equivalence between (I √ó J) by (K √ó L) matrices and J √ó I by L √ó K matrices.
