    Block Matrices from Rows and Columns #

    This file provides the basic definitions of matrices composed from columns and rows. The concatenation of two matrices with the same row indices can be expressed as A = fromCols A‚ÇÅ A‚ÇÇ the concatenation of two matrices with the same column indices can be expressed as B = fromRows B‚ÇÅ B‚ÇÇ.

    We then provide a few lemmas that deal with the products of these with each other and with block matrices
