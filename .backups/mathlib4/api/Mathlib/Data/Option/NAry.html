    Binary map of options #

    This file defines the binary map of Option. This is mostly useful to define pointwise operations on intervals.

    Main declarations #

    • Option.map‚ÇÇ: Binary map of options.

    Notes #

    This file is very similar to the n-ary section of Mathlib/Data/Set/Basic.lean, to Mathlib/Data/Finset/NAry.lean and to Mathlib/Order/Filter/NAry.lean. Please keep them in sync.

    We do not define Option.map‚ÇÉ as its only purpose so far would be to prove properties of Option.map‚ÇÇ and casing already fulfills this task.

      Algebraic replacement rules #

      A collection of lemmas to transfer associativity, commutativity, distributivity, ... of operations to the associativity, commutativity, distributivity, ... of Option.map‚ÇÇ of those operations. The proof pattern is map‚ÇÇ_lemma operation_lemma. For example, map‚ÇÇ_comm mul_comm proves that map‚ÇÇ (*) a b = map‚ÇÇ (*) g f in a CommSemigroup.
