    Prefixes, suffixes, infixes #

    This file proves properties about

    • List.isPrefix: l‚ÇÅ is a prefix of l‚ÇÇ if l‚ÇÇ starts with l‚ÇÅ.
    • List.isSuffix: l‚ÇÅ is a suffix of l‚ÇÇ if l‚ÇÇ ends with l‚ÇÅ.
    • List.isInfix: l‚ÇÅ is an infix of l‚ÇÇ if l‚ÇÅ is a prefix of some suffix of l‚ÇÇ.
    • List.inits: The list of prefixes of a list.
    • List.tails: The list of prefixes of a list.
    • insert on lists

    All those (except insert) are defined in Mathlib/Data/List/Defs.lean.

    Notation #

    • l‚ÇÅ <+: l‚ÇÇ: l‚ÇÅ is a prefix of l‚ÇÇ.
    • l‚ÇÅ <:+ l‚ÇÇ: l‚ÇÅ is a suffix of l‚ÇÇ.
    • l‚ÇÅ <:+: l‚ÇÇ: l‚ÇÅ is an infix of l‚ÇÇ.
