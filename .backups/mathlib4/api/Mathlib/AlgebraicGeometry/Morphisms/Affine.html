    Affine morphisms of schemes #

    A morphism of schemes f : X ‚ü∂ Y is affine if the preimage of an arbitrary affine open subset of Y is affine.

    It is equivalent to ask only that Y is covered by affine opens whose preimage is affine.

    Main results #

    • AlgebraicGeometry.IsAffineHom: The class of affine morphisms.
    • AlgebraicGeometry.isAffineOpen_of_isAffineOpen_basicOpen: If s is a spanning set of Œì(X, U), such that each X.basicOpen i is affine, then U is also affine.
    • AlgebraicGeometry.isAffineHom_isStableUnderBaseChange: Affine morphisms are stable under base change.

    We also provide the instance HasAffineProperty @IsAffineHom fun X _ _ _ ‚Ü¶ IsAffine X.
