    Flat morphisms #

    A morphism of schemes f : X ‚ü∂ Y is flat if for each affine U ‚äÜ Y and V ‚äÜ f ‚Åª¬π' U, the induced map Œì(Y, U) ‚ü∂ Œì(X, V) is flat. This is equivalent to asking that all stalk maps are flat (see AlgebraicGeometry.Flat.iff_flat_stalkMap).

    We show that this property is local, and are stable under compositions and base change.

      Sections of fibered products #

      Suppose we are given the following cartesian square:

      Y --g-‚Üí X
|       |
iY      iX
‚Üì       |
T --f-‚Üí S

      Let U‚Çõ be an open of S, U‚Çì and U‚Çú be opens of X and T mapping into U‚Çõ. There is a canonical map Œì(X, U‚Çì) ‚äó[Œì(S, U‚Çõ)] Œì(T, U‚Çú) ‚ü∂ Œì(X √ó‚Çõ T, pr‚ÇÅ ‚Åª¬π U‚Çì ‚à© pr‚ÇÇ ‚Åª¬π U‚Çú).

      We show that this map is

      1. isIso_pushoutSection_of_isAffineOpen: bijective when U‚Çõ, U‚Çú, and U‚Çì are all affine.
      2. mono_pushoutSection_of_isCompact_of_flat_right: injective when U‚Çõ, U‚Çú are affine, U‚Çì is compact, and f is flat.
      3. isIso_pushoutSection_of_isQuasiSeparated_of_flat_right: bijective when U‚Çõ, U‚Çú are affine, U‚Çì is qcqs, and f is flat.
      4. mono_pushoutSection_of_isCompact_of_flat_right_of_ringHomFlat: injective when U‚Çõ is affine, U‚Çú is compact, U‚Çì is qcqs, f is flat, and Œì(T, U‚Çú) is flat over Œì(S, U‚Çõ) (typically true when S = Spec k.)
      5. isIso_pushoutSection_of_isCompact_of_flat_right_of_ringHomFlat: bijective when U‚Çõ is affine, U‚Çú and U‚Çì are qcqs, f is flat, and Œì(T, U‚Çú) is flat over Œì(S, U‚Çõ) (typically true when S = Spec k.)
