    The partial recursive functions #

    The partial recursive functions are defined similarly to the primitive recursive functions, but now all functions are partial, implemented using the Part monad, and there is an additional operation, called Î¼-recursion, which performs unbounded minimization: Î¼ f returns the least natural number n for which f n = 0, or diverges if such n doesn't exist.

    Main definitions #

    • Nat.Partrec f: f is partial recursive, for functions f : â„• â†’. â„•
    • Partrec f: f is partial recursive, for partial functions between Primcodable types
    • Computable f: f is partial recursive, for total functions between Primcodable types
