    Given a language L and a word x, the left quotient of L by x is the set of suffixes y such that x ++ y is in L. The Myhillâ€“Nerode theorem shows that each left quotient, in fact, corresponds to the state of an automaton that matches L, and that L is regular if and only if there are finitely many such states.
