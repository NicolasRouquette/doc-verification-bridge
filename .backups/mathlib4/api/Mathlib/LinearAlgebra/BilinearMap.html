    Basics on bilinear maps #

    This file provides basics on bilinear maps. The most general form considered are maps that are semilinear in both arguments. They are of type M ‚Üí‚Çõ‚Çó[œÅ‚ÇÅ‚ÇÇ] N ‚Üí‚Çõ‚Çó[œÉ‚ÇÅ‚ÇÇ] P, where M and N are modules over R and S respectively, P is a module over both R‚ÇÇ and S‚ÇÇ with commuting actions, and œÅ‚ÇÅ‚ÇÇ : R ‚Üí+* R‚ÇÇ and œÉ‚ÇÅ‚ÇÇ : S ‚Üí+* S‚ÇÇ.

    Main declarations #

    • LinearMap.mk‚ÇÇ: a constructor for bilinear maps, taking an unbundled function together with proof witnesses of bilinearity
    • LinearMap.flip: turns a bilinear map M √ó N ‚Üí P into N √ó M ‚Üí P
    • LinearMap.lflip: given a linear map from M to N ‚Üí‚Çó[R] P, i.e., a bilinear map M ‚Üí N ‚Üí P, change the order of variables and get a linear map from N to M ‚Üí‚Çó[R] P.
    • LinearMap.lcomp: composition of a given linear map M ‚Üí N with a linear map N ‚Üí P as a linear map from N‚Çó ‚Üí‚Çó[R] P‚Çó to M ‚Üí‚Çó[R] P‚Çó
    • LinearMap.llcomp: composition of linear maps as a bilinear map from (M ‚Üí‚Çó[R] N) √ó (N ‚Üí‚Çó[R] P) to M ‚Üí‚Çó[R] P
    • LinearMap.compl‚ÇÇ: composition of a linear map Q ‚Üí N and a bilinear map M ‚Üí N ‚Üí P to form a bilinear map M ‚Üí Q ‚Üí P.
    • LinearMap.compr‚ÇÇ: composition of a linear map P ‚Üí Q and a bilinear map M ‚Üí N ‚Üí P to form a bilinear map M ‚Üí N ‚Üí Q.
    • LinearMap.lsmul: scalar multiplication as a bilinear map R √ó M ‚Üí M
