    Structural lemmas about finite crystallographic root pairings #

    In this file we gather basic lemmas necessary for the classification of finite crystallographic root pairings.

    Main results: #

    • RootPairing.coxeterWeightIn_mem_set_of_isCrystallographic: the Coxeter weights of a finite crystallographic root pairing belong to the set {0, 1, 2, 3, 4}.
    • RootPairing.root_sub_root_mem_of_pairingIn_pos: if Œ± ‚â† Œ≤ are both roots of a finite crystallographic root pairing, and the pairing of Œ± with Œ≤ is positive, then Œ± - Œ≤ is also a root.
    • RootPairing.root_add_root_mem_of_pairingIn_neg: if Œ± ‚â† -Œ≤ are both roots of a finite crystallographic root pairing, and the pairing of Œ± with Œ≤ is negative, then Œ± + Œ≤ is also a root.
