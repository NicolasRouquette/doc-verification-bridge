    Rank of various constructions #

    Main statements #

    • rank_quotient_add_rank_le : rank M/N + rank N ‚â§ rank M.
    • lift_rank_add_lift_rank_le_rank_prod: rank M √ó N ‚â§ rank M + rank N.
    • rank_span_le_of_finite: rank (span s) ‚â§ #s for finite s.

    For free modules, we have

    • rank_prod : rank M √ó N = rank M + rank N.
    • rank_finsupp : rank (Œπ ‚Üí‚ÇÄ M) = #Œπ * rank M
    • rank_directSum: rank (‚®Å M·µ¢) = ‚àë rank M·µ¢
    • rank_tensorProduct: rank (M ‚äó N) = rank M * rank N.

    Lemmas for ranks of submodules and subalgebras are also provided. We have finrank variants for most lemmas as well.
