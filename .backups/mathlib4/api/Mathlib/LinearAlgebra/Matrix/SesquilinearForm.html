    Sesquilinear form #

    This file defines the conversion between sesquilinear maps and matrices.

    Main definitions #

    • Matrix.toLinearMap‚ÇÇ given a basis define a bilinear map
    • Matrix.toLinearMap‚ÇÇ' define the bilinear map on n ‚Üí R
    • LinearMap.toMatrix‚ÇÇ: calculate the matrix coefficients of a bilinear map
    • LinearMap.toMatrix‚ÇÇ': calculate the matrix coefficients of a bilinear map on n ‚Üí R
