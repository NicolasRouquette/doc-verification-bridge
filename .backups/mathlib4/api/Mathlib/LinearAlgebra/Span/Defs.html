    The span of a set of vectors, as a submodule #

    • Submodule.span s is defined to be the smallest submodule containing the set s.

    Notation #

    • We introduce the notation R ‚àô v for the span of a singleton, Submodule.span R {v}. This is \span, not the same as the scalar multiplication ‚Ä¢/\bub.
