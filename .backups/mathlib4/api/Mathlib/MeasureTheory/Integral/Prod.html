    Integration with respect to the product measure #

    In this file we prove Fubini's theorem.

    Main results #

    • MeasureTheory.integrable_prod_iff states that a binary function is integrable iff both
      • y ‚Ü¶ f (x, y) is integrable for almost every x, and
      • the function x ‚Ü¶ ‚à´ ‚Äñf (x, y)‚Äñ dy is integrable.
    • MeasureTheory.integral_prod: Fubini's theorem.     • MeasureTheory.integral_integral_swap_of_hasCompactSupport: a version of Fubini's theorem for continuous functions with compact support, which does not assume that the measures are œÉ-finite contrary to all the usual versions of Fubini.
