    Circle integral transform #

    In this file we define the circle integral transform of a function f with complex domain. This is defined as $(2œÄi)^{-1}\frac{f(x)}{x-w}$ where x moves along a circle. We then prove some basic facts about these functions.

    These results are useful for proving that the uniform limit of a sequence of holomorphic functions is holomorphic.
