    For Œπ a linear order with the order topology, we define the Borel measurable space on WithTop Œπ. We then prove that the natural inclusion Œπ ‚Üí WithTop Œπ is measurable, and that the function WithTop.untopA : WithTop Œπ ‚Üí Œπ (which sends ‚ä§ to an arbitrary element of Œπ) is measurable.

    Main statements #

    • Measurable.untopD: for d : Œπ, the function WithTop.untopD d : WithTop Œπ ‚Üí Œπ is measurable.
