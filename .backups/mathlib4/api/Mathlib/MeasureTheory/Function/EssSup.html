    Essential supremum and infimum #

    We define the essential supremum and infimum of a function f : Œ± ‚Üí Œ≤ with respect to a measure Œº on Œ±. The essential supremum is the infimum of the constants c : Œ≤ such that f x ‚â§ c almost everywhere.

    There is a different quantity which is sometimes also called essential supremum: the least upper-bound among measurable functions of a family of measurable functions (in an almost-everywhere sense). We do not define that quantity here, which is simply the supremum of a map with values in Œ± ‚Üí‚Çò[Œº] Œ≤ (see Mathlib/MeasureTheory/Function/AEEqFun.lean).
