    F√∏lner sequences and filters - definitions and properties #

    This file defines F√∏lner sequences and filters for measurable spaces acted on by a group.

    Definitions #

    • IsFoelner G Œº l F : Consider a group G acting on a measure space X. A sequence of sets F : Œπ ‚Üí Set X is F√∏lner with respect to the G-action, the measure Œº, and a filter l on the indexing type Œπ, if:

      1. Eventually, as i tends to l, the set F i is measurable with finite non-zero measure,
      2. For all g : G, Œº ((g ‚Ä¢ F i) ‚àÜ F i) / Œº (F i) tends to 0.

    • IsFoelner.mean Œº u F s : The limit along an ultrafilter u of the density of a set s with respect to a F√∏lner sequence F in the measure space X.

    • maxFoelner G Œº : The maximal F√∏lner filter with respect to some group G acting on a measure space X is the pullback of ùìù 0 along the map s ‚Ü¶ Œº (g ‚Ä¢ s) / Œº s over measurable sets of finite non-zero measure.

    • IsAddFoelner G Œº l F: the analog of IsFoelner G Œº l F for an additive group action

    Main results #

    • IsFoelner.amenable : If there exists a non-trivial F√∏lner filter with respect to some group G acting on a measure space X, then there exists a G-invariant finitely additive probability measure on X.

    • isFoelner_iff_tendsto : A sequence of sets is F√∏lner if and only if it tends to the maximal F√∏lner filter. The attribute "maximal" of the latter comes from the direct implication of this theorem : if IsFoelner G Œº l F then the push-forward filter map F l ‚â§ maxFoelner G Œº.

    • amenable_of_maxFoelner_neBot : If the maximal F√∏lner filter is non-trivial, then there exists a G-invariant finitely additive probability measure on X.

    Temporary design adaptations #

    • In the current version, we refer to the amenability of the action of a group on a measure space (e.g. in IsFoelner.amenable and amenable_of_maxFoelner_neBot), even though a definition of amenability has not yet been given in Mathlib. This is because there are different notions of amenability for groups and for group actions, and a Mathlib definition should be provided at the greatest level of generality, on which there has not yet been a general consensus. At the present moment, amenable corresponds to the existence of a G-invariant finitely additive probability measure.
