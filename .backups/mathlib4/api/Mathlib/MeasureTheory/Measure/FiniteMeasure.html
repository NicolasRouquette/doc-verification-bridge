    Finite measures #

    This file defines the type of finite measures on a given measurable space. When the underlying space has a topology and the measurable space structure (sigma algebra) is finer than the Borel sigma algebra, then the type of finite measures is equipped with the topology of weak convergence of measures. The topology of weak convergence is the coarsest topology w.r.t. which for every bounded continuous ‚Ñù‚â•0-valued function f, the integration of f against the measure is continuous.

    Main results #

    • Finite measures Œº on Œ© give rise to continuous linear functionals on the space of bounded continuous nonnegative functions on Œ© via integration: MeasureTheory.FiniteMeasure.toWeakDualBCNN : FiniteMeasure Œ© ‚Üí (WeakDual ‚Ñù‚â•0 (Œ© ‚Üí·µá ‚Ñù‚â•0))

    Implementation notes #

    The topology of weak convergence of finite Borel measures is defined using a mapping from MeasureTheory.FiniteMeasure Œ© to WeakDual ‚Ñù‚â•0 (Œ© ‚Üí·µá ‚Ñù‚â•0), inheriting the topology from the latter.

    The implementation of MeasureTheory.FiniteMeasure Œ© and is directly as a subtype of MeasureTheory.Measure Œ©, and the coercion to a function is the composition ENNReal.toNNReal and the coercion to function of MeasureTheory.Measure Œ©. Another alternative would have been to use a bijection with MeasureTheory.VectorMeasure Œ© ‚Ñù‚â•0 as an intermediate step. Some considerations:

    • Potential advantages of using the NNReal-valued vector measure alternative:
      • The coercion to function would avoid need to compose with ENNReal.toNNReal, the NNReal-valued API could be more directly available.
    • Potential drawbacks of the vector measure alternative:
      • The coercion to function would lose monotonicity, as non-measurable sets would be defined to have measure 0.
      • No integration theory directly. E.g., the topology definition requires MeasureTheory.lintegral w.r.t. a coercion to MeasureTheory.Measure Œ© in any case.
