    Contents #

    In this file we work with contents. A content Œª is a function from a certain class of subsets (such as the compact subsets) to ‚Ñù‚â•0 that is

    • additive: If K‚ÇÅ and K‚ÇÇ are disjoint sets in the domain of Œª, then Œª(K‚ÇÅ ‚à™ K‚ÇÇ) = Œª(K‚ÇÅ) + Œª(K‚ÇÇ);
    • subadditive: If K‚ÇÅ and K‚ÇÇ are in the domain of Œª, then Œª(K‚ÇÅ ‚à™ K‚ÇÇ) ‚â§ Œª(K‚ÇÅ) + Œª(K‚ÇÇ);
    • monotone: If K‚ÇÅ ‚äÜ K‚ÇÇ are in the domain of Œª, then Œª(K‚ÇÅ) ‚â§ Œª(K‚ÇÇ).

    We show that:

    • Given a content Œª on compact sets, let us define a function Œª* on open sets, by letting Œª* U be the supremum of Œª K for K included in U. This is a countably subadditive map that vanishes at ‚àÖ. In Halmos (1950) this is called the inner content Œª* of Œª, and formalized as innerContent.
    • Given an inner content, we define an outer measure Œº*, by letting Œº* E be the infimum of Œª* U over the open sets U containing E. This is indeed an outer measure. It is formalized as outerMeasure.
    • Restricting this outer measure to Borel sets gives a regular measure Œº.

    We define bundled contents as Content. In this file we only work on contents on compact sets, and inner contents on open sets, and both contents and inner contents map into the extended nonnegative reals. However, in other applications other choices can be made, and it is not a priori clear what the best interface should be.

    Main definitions #

    For Œº : Content G, we define

    • Œº.innerContent : the inner content associated to Œº.
    • Œº.outerMeasure : the outer measure associated to Œº.
    • Œº.measure : the Borel measure associated to Œº.

    These definitions are given for spaces which are R‚ÇÅ. The resulting measure Œº.measure is always outer regular by design. When the space is locally compact, Œº.measure is also regular.
