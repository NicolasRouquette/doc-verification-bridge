    Miscellaneous arithmetic Functions #

    This file defines some simple examples of arithmetic functions (functions ‚Ñï ‚Üí R vanishing at 0, considered as a ring under Dirichlet convolution). Note that the Von Mangoldt and M√∂bius functions are in separate files.

    Main Definitions #

    • œÉ k is the arithmetic function such that œÉ k x = ‚àë y ‚àà divisors x, y ^ k for 0 < x.
    • pow k is the arithmetic function such that pow k x = x ^ k for 0 < x.
    • id is the identity arithmetic function on ‚Ñï.
    • œâ n is the number of distinct prime factors of n.
    • Œ© n is the number of prime factors of n counted with multiplicity.

    Notation #

    The arithmetic functions œÉ, œâ and Œ© have Greek letter names. This notation is scoped to the separate locales ArithmeticFunction.sigma for œÉ, ArithmeticFunction.omega for œâ and ArithmeticFunction.Omega for Œ©, to allow for selective access.
