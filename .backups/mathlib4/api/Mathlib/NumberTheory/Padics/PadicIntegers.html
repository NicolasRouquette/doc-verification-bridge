    p-adic integers #

    This file defines the p-adic integers ‚Ñ§_[p] as the subtype of ‚Ñö_[p] with norm ‚â§ 1. We show that ‚Ñ§_[p]

    • is complete,
    • is nonarchimedean,
    • is a normed ring,
    • is a local ring, and
    • is a discrete valuation ring.

    The relation between ‚Ñ§_[p] and ZMod p is established in another file.

    Important definitions #

    • PadicInt : the type of p-adic integers

    Notation #

    We introduce the notation ‚Ñ§_[p] for the p-adic integers.

    Implementation notes #

    Much, but not all, of this file assumes that p is prime. This assumption is inferred automatically by taking [Fact p.Prime] as a type class argument.

    Coercions into ‚Ñ§_[p] are set up to work with the norm_cast tactic.

    • [R. Y. Lewis, A formal proof of Hensel's lemma over the p-adic integers][lewis2019]
