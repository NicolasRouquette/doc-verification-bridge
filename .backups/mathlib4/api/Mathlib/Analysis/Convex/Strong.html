    Uniformly and strongly convex functions #

    In this file, we define uniformly convex functions and strongly convex functions.

    For a real normed space E, a uniformly convex function with modulus œÜ : ‚Ñù ‚Üí ‚Ñù is a function f : E ‚Üí ‚Ñù such that f (t ‚Ä¢ x + (1 - t) ‚Ä¢ y) ‚â§ t ‚Ä¢ f x + (1 - t) ‚Ä¢ f y - t * (1 - t) * œÜ ‚Äñx - y‚Äñ for all t ‚àà [0, 1].
