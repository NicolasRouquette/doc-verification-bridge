    (Local) maximums in a normed space #

    In this file we prove the following lemma, see IsMaxFilter.norm_add_sameRay. If f : Œ± ‚Üí E is a function such that norm ‚àò f has a maximum along a filter l at a point c and y is a vector on the same ray as f c, then the function fun x => ‚Äñf x + y‚Äñ has a maximum along l at c.

    Then we specialize it to the case y = f c and to different special cases of IsMaxFilter: IsMaxOn, IsLocalMaxOn, and IsLocalMax.
