    Type synonym for types with a CStarModule structure #

    It is often the case that we want to construct a CStarModule instance on a type that is already endowed with a norm, but this norm is not the one associated to its CStarModule structure. For this reason, we create a type synonym WithCStarModule which is endowed with the requisite CStarModule instance. We also introduce the scoped notation C‚ãÜ·µê·µí·µà for this type synonym.

    The common use cases are, when A is a C‚ãÜ-algebra:

    • E √ó F where E and F are CStarModules over A
    • Œ† i, E i where E i is a CStarModule over A and i : Œπ with Œπ a Fintype

    In this way, the set up is very similar to the WithLp type synonym, although there is no way to reuse WithLp because the norms do not coincide in general.

    The WithCStarModule synonym is of vital importance, especially because the CStarModule class marks A as an outParam. Indeed, we want to infer A from the type of E, but, as with modules, a type E can be a CStarModule over different C‚ãÜ-algebras. For example, note that if A is a C‚ãÜ-algebra, then so is A √ó A, and therefore we may consider both A and A √ó A as CStarModules over themselves, respectively. However, we may also consider A √ó A as a CStarModule over A. However, by utilizing the type synonym, these actually correspond to different types, namely:

    • A as a CStarModule over A corresponds to A
    • A √ó A as a CStarModule over A √ó A corresponds to A √ó A
    • A √ó A as a CStarModule over A corresponds to C‚ãÜ·µê·µí·µà (A √ó A)

    Main definitions #

    • WithCStarModule A E: a copy of E to be equipped with a CStarModule A structure.
    • WithCStarModule.equiv A E: the canonical equivalence between WithCStarModule A E and E.
    • WithCStarModule.linearEquiv ‚ÑÇ A E: the canonical ‚ÑÇ-module isomorphism between WithCStarModule A E and E.

    Implementation notes #

    The pattern here is the same one as is used by Lex for order structures; it avoids having a separate synonym for each type, and allows all the structure-copying code to be shared.
