    Isometric continuous functional calculus #

    This file adds a class for an isometric continuous functional calculus. This is separate from the usual ContinuousFunctionalCalculus class because we prefer not to require a metric (or a norm) on the algebra for reasons discussed in the module documentation for that file.

    Of course, with a metric on the algebra and an isometric continuous functional calculus, the algebra must be a Câ‹†-algebra already. As such, it may seem like this class is not useful. However, the main purpose is to allow for the continuous functional calculus to be an isometry for the other scalar rings â„ and â„â‰¥0 too.
