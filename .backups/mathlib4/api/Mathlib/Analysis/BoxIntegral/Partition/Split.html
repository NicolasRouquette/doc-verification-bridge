    Split a box along one or more hyperplanes #

    Main definitions #

    A hyperplane {x : Œπ ‚Üí ‚Ñù | x i = a} splits a rectangular box I : BoxIntegral.Box Œπ into two smaller boxes. If a ‚àâ Ioo (I.lower i, I.upper i), then one of these boxes is empty, so it is not a box in the sense of BoxIntegral.Box.

    We introduce the following definitions.

    • BoxIntegral.Box.splitLower I i a and BoxIntegral.Box.splitUpper I i a are these boxes (as WithBot (BoxIntegral.Box Œπ));
    • BoxIntegral.Prepartition.split I i a is the partition of I made of these two boxes (or of one box I if one of these boxes is empty);
    • BoxIntegral.Prepartition.splitMany I s, where s : Finset (Œπ √ó ‚Ñù) is a finite set of hyperplanes {x : Œπ ‚Üí ‚Ñù | x i = a} encoded as pairs (i, a), is the partition of I made by cutting it along all the hyperplanes in s.

    Main results #

    The main result BoxIntegral.Prepartition.exists_iUnion_eq_diff says that any prepartition œÄ of I admits a prepartition œÄ' of I that covers exactly I \ œÄ.iUnion. One of these prepartitions is available as BoxIntegral.Prepartition.compl.
