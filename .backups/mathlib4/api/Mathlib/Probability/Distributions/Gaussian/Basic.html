    Gaussian distributions in Banach spaces #

    We introduce a predicate IsGaussian for measures on a Banach space E such that the map by any continuous linear form is a Gaussian measure on ‚Ñù.

    For Gaussian distributions in ‚Ñù, see the file Mathlib/Probability/Distributions/Gaussian/Real.lean.

    Main definitions #

    • IsGaussian: a measure Œº is Gaussian if its map by every continuous linear form L : Dual ‚Ñù E is a real Gaussian measure. That is, Œº.map L = gaussianReal (Œº[L]) (Var[L; Œº]).toNNReal.

    Main statements #

    • isGaussian_iff_charFunDual_eq: a finite measure Œº is Gaussian if and only if its characteristic function has value exp (Œº[L] * I - Var[L; Œº] / 2) for every continuous linear form L : Dual ‚Ñù E.
