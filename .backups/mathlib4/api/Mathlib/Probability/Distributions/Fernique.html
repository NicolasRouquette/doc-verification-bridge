    Fernique's theorem for rotation-invariant measures #

    Let Œº be a finite measure on a second-countable normed space E such that the product measure Œº.prod Œº on E √ó E is invariant by rotation of angle -œÄ/4. Then there exists a constant C > 0 such that the function x ‚Ü¶ exp (C * ‚Äñx‚Äñ ^ 2) is integrable with respect to Œº.

    Sketch of the proof #

    The main case of the proof is for Œº a probability measure such that there exists a positive a : ‚Ñù such that 2‚Åª¬π < Œº {x | ‚Äñx‚Äñ ‚â§ a} < 1. If Œº is a probability measure and a does not exist then we can show that there is a ball with finite radius of measure 1, and the result is true for C = 1 (for example), since x ‚Ü¶ exp (‚Äñx‚Äñ ^ 2) is almost surely bounded. We then choose such an a.

    In order to show the existence of C such that x ‚Ü¶ exp (C * ‚Äñx‚Äñ ^ 2) is integrable, we prove as intermediate result that for a, c with 2‚Åª¬π < c ‚â§ Œº {x | ‚Äñx‚Äñ ‚â§ a}, the integral ‚à´‚Åª x, exp (logRatio c * a‚Åª¬π ^ 2 * ‚Äñx‚Äñ ^ 2) ‚àÇŒº is bounded by a finite quantity (logRatio c is a multiple of log (c / (1 - c))). We can then take C = logRatio c * a‚Åª¬π ^ 2.

    We now turn to the proof of the intermediate result.

    First in measure_le_mul_measure_gt_le_of_map_rotation_eq_self we prove that if a measure Œº is such that Œº.prod Œº is invariant by rotation of angle -œÄ/4 then Œº {x | ‚Äñx‚Äñ ‚â§ a} * Œº {x | b < ‚Äñx‚Äñ} ‚â§ Œº {x | (b - a) / ‚àö2 < ‚Äñx‚Äñ} ^ 2. The rotation invariance is used only through that inequality.

    We define a sequence of thresholds t n inductively by t 0 = a and t (n + 1) = ‚àö2 * t n + a. They are chosen such that the invariance by rotation gives Œº {x | ‚Äñx‚Äñ ‚â§ a} * Œº {x | t (n + 1) < ‚Äñx‚Äñ} ‚â§ Œº {x | t n < ‚Äñx‚Äñ} ^ 2. Thanks to that inequality we can show that Œº {x | t n < ‚Äñx‚Äñ} decreases fast with n: for m‚Çê = Œº {x | ‚Äñx‚Äñ ‚â§ a}, Œº {x | t n < ‚Äñx‚Äñ} ‚â§ m‚Çê * exp (- log (m‚Çê / (1 - m‚Çê)) * 2 ^ n).

    We cut the space into annuli {x | t n < ‚Äñx‚Äñ ‚â§ t n + 1} and bound the integral separately on each annulus. On that set the function exp (logRatio c * a‚Åª¬π ^ 2 * ‚Äñx‚Äñ ^ 2) is bounded by exp (logRatio c * a‚Åª¬π ^ 2 * t (n + 1) ^ 2), which is in turn less than exp (2‚Åª¬π * log (c / (1 - c)) * 2 ^ n) (from the definition of the threshold t and logRatio c). The measure of the annulus is bounded by Œº {x | t n < ‚Äñx‚Äñ}, for which we derived an upper bound above. The function gets exponentially large, but Œº {x | t n < ‚Äñx‚Äñ} decreases even faster, so the integral is bounded by a quantity of the form exp (- u * 2 ^ n) for u>0. Summing over all annuli (over n) gives a finite value for the integral.
