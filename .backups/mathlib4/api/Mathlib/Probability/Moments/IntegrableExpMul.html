    Domain of the moment-generating function #

    For X a real random variable and Œº a finite measure, the set {t | Integrable (fun œâ ‚Ü¶ exp (t * X œâ)) Œº} is an interval containing zero. This is the set of points for which the moment-generating function mgf X Œº t is well defined. We denote that set by integrableExpSet X Œº.

    We prove the integrability of other functions for t in the interior of that interval.

    Main definitions #

    • ProbabilityTheory.IntegrableExpSet: the interval of reals for which exp (t * X) is integrable.

    Main results #

    • ProbabilityTheory.integrable_exp_mul_of_le_of_le: if exp (u * X) is integrable for u = a and u = b then it is integrable on [a, b].
    • ProbabilityTheory.convex_integrableExpSet: integrableExpSet X Œº is a convex set.
    • ProbabilityTheory.integrable_exp_mul_of_nonpos_of_ge: if exp (u * X) is integrable for u ‚â§ 0 then it is integrable on [u, 0].
    • ProbabilityTheory.integrable_rpow_abs_mul_exp_of_mem_interior: for v in the interior of the interval in which exp (t * X) is integrable, for all nonnegative p : ‚Ñù, |X| ^ p * exp (v * X) is integrable.
    • ProbabilityTheory.memLp_of_mem_interior_integrableExpSet: if 0 belongs to the interior of integrableExpSet X Œº, then X is in ‚Ñíp for all finite p.
