    Cantor Normal Form #

    The Cantor normal form of an ordinal is generally defined as its base œâ expansion, with its non-zero exponents in decreasing order. Here, we more generally define a base b expansion Ordinal.CNF in this manner, which is well-behaved for any b ‚â• 2.

    Implementation notes #

    We implement Ordinal.CNF as an association list, where keys are exponents and values are coefficients. This is because this structure intrinsically reflects two key properties of the Cantor normal form:

    • It is ordered.
    • It has finitely many entries.
