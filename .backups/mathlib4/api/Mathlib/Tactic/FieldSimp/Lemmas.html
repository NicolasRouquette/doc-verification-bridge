    This is a variant of integer exponentiation, defined for internal use in the field_simp tactic implementation. It differs from the usual integer exponentiation in that 0 ^ 0 is 0, not 1. With this choice, the function n ‚Ü¶ a ^ n is always a homomorphism (a ^ (n + m) = a ^ n * a ^ m), even if a is zero.

      Theory of lists of pairs (exponent, atom) #

      This section contains the lemmas which are orchestrated by the field_simp tactic to prove goals in fields. The basic object which these lemmas concern is NF M, a type synonym for a list of ordered pairs in ‚Ñ§ √ó M, where typically M is a field.
