    Local properties of commutative rings #

    In this file, we define local properties in general.

    Naming Conventions #

    • localization_P : P holds for S‚Åª¬πR if P holds for R.
    • P_of_localization_maximal : P holds for R if P holds for R‚Çò for all maximal m.
    • P_of_localization_prime : P holds for R if P holds for R‚Çò for all prime m.
    • P_ofLocalizationSpan : P holds for R if given a spanning set {f·µ¢}, P holds for all R_{f·µ¢}.

    Main definitions #

    • LocalizationPreserves : A property P of comm rings is said to be preserved by localization if P holds for M‚Åª¬πR whenever P holds for R.
    • OfLocalizationMaximal : A property P of comm rings satisfies OfLocalizationMaximal if P holds for R whenever P holds for R‚Çò for all maximal ideal m.
    • RingHom.LocalizationPreserves : A property P of ring homs is said to be preserved by localization if P holds for M‚Åª¬πR ‚Üí+* M‚Åª¬πS whenever P holds for R ‚Üí+* S.
    • RingHom.OfLocalizationSpan : A property P of ring homs satisfies RingHom.OfLocalizationSpan if P holds for R ‚Üí+* S whenever there exists a set { r } that spans R such that P holds for R·µ£ ‚Üí+* S·µ£.

    Main results #

    • The triviality of an ideal or an element: ideal_eq_bot_of_localization, eq_zero_of_localization
