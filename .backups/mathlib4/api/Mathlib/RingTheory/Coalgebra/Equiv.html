    Isomorphisms of R-coalgebras #

    This file defines bundled isomorphisms of R-coalgebras. We largely mirror the basic API of Mathlib/Algebra/Module/Equiv/Defs.lean.

    Main definitions #

    • CoalgEquiv R A B: the type of R-coalgebra isomorphisms between A and B.
