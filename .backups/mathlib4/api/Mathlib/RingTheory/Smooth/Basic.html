    Smooth morphisms #

    An R-algebra A is formally smooth if Œ©[A‚ÅÑR] is A-projective and H¬π(L_{A/R}) = 0. This is equivalent to the standard definition that "for every R-algebra B, every square-zero ideal I : Ideal B and f : A ‚Üí‚Çê[R] B ‚ß∏ I, there exists at least one lift A ‚Üí‚Çê[R] B". An R-algebra A is smooth if it is formally smooth and of finite presentation.

    We show that the property of being formally smooth extends onto nilpotent ideals, and that it is stable under R-algebra homomorphisms and compositions.

    We show that smooth is stable under algebra isomorphisms, composition and localization at an element.

    Main results #

    • Algebra.FormallySmooth: The class of formally smooth algebras.
    • Algebra.formallySmooth_iff : Formally smooth iff Œ©[A‚ÅÑR] is A-projective and H¬π(L_{A/R}) = 0.
    • Algebra.FormallySmooth.lift: If A is formally smooth and I is nilpotent, any map A ‚Üí‚Çê[R] B ‚ß∏ I lifts to A ‚Üí‚Çê[R] B.
    • Algebra.FormallySmooth.iff_comp_surjective: A is formally smooth iff any map A ‚Üí‚Çê[R] B ‚ß∏ I lifts to A ‚Üí‚Çê[R] B for any square zero I.

    Suppose P is a formally smooth R algebra that surjects onto A with kernel I, then

    • Algebra.FormallySmooth.iff_split_surjection: A is formally smooth iff the algebra map P ‚ß∏ I¬≤ ‚Üí‚Çê[R] A has an R-algebra section.
    • Algebra.Extension.equivH1CotangentOfFormallySmooth: H¬π(L_{A/R}) is isomorphic to ker(I/I¬≤ ‚Üí A ‚äó[P] Œ©[P‚ÅÑR]).
    • Algebra.FormallySmooth.iff_split_injection: A is formally smooth iff the P-linear map I/I¬≤ ‚Üí A ‚äó[P] Œ©[P‚ÅÑR] is split injective.
