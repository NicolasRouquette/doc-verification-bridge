    Extension of fractional ideals #

    This file defines the extension of a fractional ideal along a ring homomorphism.

    Main definitions #

    • FractionalIdeal.extended: Let A and B be commutative rings with respective localizations IsLocalization M K and IsLocalization N L. Let f : A ‚Üí+* B be a ring homomorphism with hf : M ‚â§ Submonoid.comap f N. If I : FractionalIdeal M K, then the extension of I along f is extended L hf I : FractionalIdeal N L.
    • FractionalIdeal.extendedHom: The ring homomorphism version of FractionalIdeal.extended.
    • FractionalIdeal.extendedHom‚Çê: For A ‚äÜ B an extension of domains, the ring homomorphism that sends a fractional ideal of A to a fractional ideal of B.

    Main results #

    • FractionalIdeal.extendedHom‚Çê_injective: the map FractionalIdeal.extendedHom‚Çê is injective.
    • Ideal.map_algebraMap_injective: For A ‚äÜ B an extension of Dedekind domains, the map that sends an ideal I of A to I¬∑B is injective.
