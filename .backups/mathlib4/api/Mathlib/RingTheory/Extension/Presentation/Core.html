    Presentations on subrings #

    In this file we establish the API for realising a presentation over a subring of R. We define a property HasCoeffs R‚ÇÄ for a presentation P to mean the (sub)ring R‚ÇÄ contains the coefficients of the relations of P. subring R‚ÇÄ of R that contains the coefficients of the relations In this case there exists a model of S over R‚ÇÄ, i.e., there exists an R‚ÇÄ-algebra S‚ÇÄ such that S is isomorphic to R ‚äó[R‚ÇÄ] S‚ÇÄ.

    If the presentation is finite, R‚ÇÄ may be chosen as a Noetherian ring. In this case, this API can be used to remove Noetherian hypothesis in certain cases.
