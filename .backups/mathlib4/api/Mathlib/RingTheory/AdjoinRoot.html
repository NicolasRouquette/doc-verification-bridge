    Adjoining roots of polynomials #

    This file defines the commutative ring AdjoinRoot f, the ring R[X]/(f) obtained from a commutative ring R and a polynomial f : R[X]. If furthermore R is a field and f is irreducible, the field structure on AdjoinRoot f is constructed.

    We suggest stating results on IsAdjoinRoot instead of AdjoinRoot to achieve higher generality, since IsAdjoinRoot works for all different constructions of R[Œ±] including AdjoinRoot f = R[X]/(f) itself.

    Main definitions and results #

    The main definitions are in the AdjoinRoot namespace.

    • mk f : R[X] ‚Üí+* AdjoinRoot f, the natural ring homomorphism.

    • of f : R ‚Üí+* AdjoinRoot f, the natural ring homomorphism.

    • root f : AdjoinRoot f, the image of X in R[X]/(f).

    • lift (i : R ‚Üí+* S) (x : S) (h : f.eval‚ÇÇ i x = 0) : (AdjoinRoot f) ‚Üí+* S, the ring homomorphism from R[X]/(f) to S extending i : R ‚Üí+* S and sending X to x.

    • lift_hom (x : S) (hfx : aeval x f = 0) : AdjoinRoot f ‚Üí‚Çê[R] S, the algebra homomorphism from R[X]/(f) to S extending algebraMap R S and sending X to x

    • equiv : (AdjoinRoot f ‚Üí‚Çê[F] E) ‚âÉ {x // x ‚àà f.aroots E} a bijection between algebra homomorphisms from AdjoinRoot and roots of f in S
