    Target local closure of ring homomorphism properties #

    If P is a property of ring homomorphisms, we call Locally P the closure of P with respect to standard open coverings on the (algebraic) target (i.e. geometric source). Hence for f : R ‚Üí+* S, the property Locally P holds if it holds locally on S, i.e. if there exists a subset { t } of S generating the unit ideal, such that P holds for all compositions R ‚Üí+* S‚Çú.

    Assuming without further mention that P is stable under composition with isomorphisms, Locally P is local on the target by construction, i.e. it satisfies RingHom.OfLocalizationSpanTarget. If P itself is local on the target, Locally P coincides with P.

    The Locally construction preserves various properties of P, e.g. if P is stable under composition, base change, etc., so is Locally P.

    Main results #

    • RingHom.locally_ofLocalizationSpanTarget: Locally P is local on the target.
    • RingHom.locally_holdsForLocalizationAway: Locally P holds for localization away maps if P does.
    • RingHom.locally_isStableUnderBaseChange: Locally P is stable under base change if P is.
    • RingHom.locally_stableUnderComposition: Locally P is stable under composition if P is and P is preserved under localizations.
    • RingHom.locally_stableUnderCompositionWithLocalizationAwayTarget and RingHom.locally_stableUnderCompositionWithLocalizationAwaySource: Locally P is stable under composition with localization away maps if P is.
    • RingHom.locally_localizationPreserves: If P is preserved by localizations, then so is Locally P.
