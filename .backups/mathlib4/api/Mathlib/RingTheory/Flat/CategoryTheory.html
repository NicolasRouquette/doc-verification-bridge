    Tensoring with a flat module is an exact functor #

    In this file we prove that tensoring with a flat module is an exact functor.

    Main results #

    • Module.Flat.iff_lTensor_preserves_shortComplex_exact: an R-module M is flat if and only if for every exact sequence A ‚ü∂ B ‚ü∂ C, M ‚äó A ‚ü∂ M ‚äó B ‚ü∂ M ‚äó C is also exact.

    • Module.Flat.iff_rTensor_preserves_shortComplex_exact: an R-module M is flat if and only if for every short exact sequence A ‚ü∂ B ‚ü∂ C, A ‚äó M ‚ü∂ B ‚äó M ‚ü∂ C ‚äó M is also exact.
