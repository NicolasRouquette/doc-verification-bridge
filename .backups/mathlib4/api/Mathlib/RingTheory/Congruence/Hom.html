    Congruence relations and ring homomorphisms #

    This file contains elementary definitions involving congruence relations and morphisms for rings and semirings

    Main definitions #

    • RingCon.ker: the kernel of a monoid homomorphism as a congruence relation

    • RingCon.lift, RingCon.lift‚Çê: the homomorphism / the algebra morphism on the quotient given that the congruence is in the kernel

    • RingCon.map, RingCon.map‚Çê: homomorphism / algebra morphism from a smaller to a larger quotient

    • RingCon.quotientKerEquivRangeS, RingCon.quotientKerEquivRange, RingCon.quotientKerEquivRange‚Çê : the first isomorphism theorem for semirings (using RingHom.rangeS), rings (using RingHom.range) and algebras (using AlgHom.range).

    • RingCon.comapQuotientEquivRangeS, RingCon.comapQuotientEquivRange, RingCon.comapQuotientEquivRange‚Çê : the second isomorphism theorem for semirings (using RingHom.rangeS), rings (using RingHom.range) and algebras (using AlgHom.range).

    • RingCon.quotientQuotientEquivQuotient, RingCon.quotientQuotientEquivQuotient‚Çê : the third isomorphism theorem for semirings (or rings) and algebras
