    Intervals Ixx (f x) (f (Order.succ x)) #

    In this file we prove

    • Monotone.biUnion_Ico_Ioc_map_succ: if Œ± is a linear archimedean succ order and Œ≤ is a linear order, then for any monotone function f and m n : Œ±, the union of intervals Set.Ioc (f i) (f (Order.succ i)), m ‚â§ i < n, is equal to Set.Ioc (f m) (f n);

    • Monotone.pairwise_disjoint_on_Ioc_succ: if Œ± is a linear succ order, Œ≤ is a preorder, and f : Œ± ‚Üí Œ≤ is a monotone function, then the intervals Set.Ioc (f n) (f (Order.succ n)) are pairwise disjoint.

    For the latter lemma, we also prove various order dual versions.
