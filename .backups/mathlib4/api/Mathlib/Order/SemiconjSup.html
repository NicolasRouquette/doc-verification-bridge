    Semiconjugate by sSup #

    In this file we prove two facts about semiconjugate (families of) functions.

    First, if an order isomorphism fa : Œ± ‚Üí Œ± is semiconjugate to an order embedding fb : Œ≤ ‚Üí Œ≤ by g : Œ± ‚Üí Œ≤, then fb is semiconjugate to fa by y ‚Ü¶ sSup {x | g x ‚â§ y}, see Semiconj.symm_adjoint.
