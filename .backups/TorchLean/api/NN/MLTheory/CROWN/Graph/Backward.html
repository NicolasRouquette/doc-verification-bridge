    CROWN Graph Backward Objective API #

    Public entrypoint for objective-dependent backward CROWN bounds. The implementation lives with the forward affine engine in Graph/Engine, because the backward pass reuses the same affine-transfer machinery. This module gives users a focused import for the backward chapter without exposing those implementation helpers as public API.
