    Functions on Space d which can be made into distributions #

    i. Overview #

    In this module, for functions f : Space d ‚Üí F, we define the property IsDistBounded f. Functions satisfying this property can be used to create distributions Space d ‚Üíd[‚Ñù] F by integrating them against Schwartz maps.
