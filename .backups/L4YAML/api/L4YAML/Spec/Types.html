    YAML Value Types #

    Core AST types for representing YAML documents.

    These types are intentionally identical to LeanYaml.YamlValue from lean4-yaml, enabling the Schema/FromToYaml/Deriving/Emitter layers to be shared between the original (unverified) and verified parser implementations.
