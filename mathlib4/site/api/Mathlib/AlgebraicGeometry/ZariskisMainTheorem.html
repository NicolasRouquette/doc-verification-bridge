    Zariski's Main Theorem #

    In this file we prove Grothendieck's reformulation of Zariski's main theorem, namely if f : X ‚ü∂ Y is separated and of finite type, then the map from the quasi-finite locus U ‚äÜ X of f to the relative normalization X' of Y in X is an open immersion.

    We then have the following corollaries

    • Scheme.Hom.isOpen_quasiFiniteAt : If f is separated and of finite type, then the quasi-finite locus of f is open.
    • If f is itself quasi-finite, then the map f.toNormalization : X ‚ü∂ X' is an open immersion. This can be accessed via inferInstance.
    • IsFinite.of_isProper_of_locallyQuasiFinite: If f is proper and quasi-finite, then the map f.toNormalization : X ‚ü∂ X' is an isomorphism, which implies that f itself is finite.
