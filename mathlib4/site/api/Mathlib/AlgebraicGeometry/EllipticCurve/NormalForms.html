    Some normal forms of elliptic curves #

    This file defines some normal forms of Weierstrass equations of elliptic curves.

    Main definitions and results #

    The following normal forms are in [silverman2009], section III.1, page 42.

    • WeierstrassCurve.IsCharNeTwoNF is a type class which asserts that a WeierstrassCurve is of form Y¬≤ = X¬≥ + a‚ÇÇX¬≤ + a‚ÇÑX + a‚ÇÜ. It is the normal form of characteristic ‚â† 2.

      If 2 is invertible in the ring (for example, if it is a field of characteristic ‚â† 2), then for any WeierstrassCurve there exists a change of variables which will change it into such normal form (WeierstrassCurve.exists_variableChange_isCharNeTwoNF). See also WeierstrassCurve.toCharNeTwoNF and WeierstrassCurve.toCharNeTwoNF_spec.

    The following normal forms are in [silverman2009], Appendix A, Proposition 1.1.

    • WeierstrassCurve.IsShortNF is a type class which asserts that a WeierstrassCurve is of form Y¬≤ = X¬≥ + a‚ÇÑX + a‚ÇÜ. It is the normal form of characteristic ‚â† 2 or 3, and also the normal form of characteristic = 3 and j = 0.

      If 2 and 3 are invertible in the ring (for example, if it is a field of characteristic ‚â† 2 or 3), then for any WeierstrassCurve there exists a change of variables which will change it into such normal form (WeierstrassCurve.exists_variableChange_isShortNF).       If the ring is of characteristic = 3, then for any WeierstrassCurve with b‚ÇÇ = 0 (for an elliptic curve, this is equivalent to j = 0), there exists a change of variables which will change it into such normal form (see WeierstrassCurve.toShortNFOfCharThree and WeierstrassCurve.toShortNFOfCharThree_spec).

    • WeierstrassCurve.IsCharThreeJNeZeroNF is a type class which asserts that a WeierstrassCurve is of form Y¬≤ = X¬≥ + a‚ÇÇX¬≤ + a‚ÇÜ. It is the normal form of characteristic = 3 and j ‚â† 0.

      If the field is of characteristic = 3, then for any WeierstrassCurve with b‚ÇÇ ‚â† 0 (for an elliptic curve, this is equivalent to j ‚â† 0), there exists a change of variables which will change it into such normal form (see WeierstrassCurve.toCharThreeNF and WeierstrassCurve.toCharThreeNF_spec_of_b‚ÇÇ_ne_zero).

      If the ring is of characteristic = 2, then for any WeierstrassCurve with a‚ÇÅ = 0 (for an elliptic curve, this is equivalent to j = 0), there exists a change of variables which will change it into such normal form (see WeierstrassCurve.toCharTwoJEqZeroNF and WeierstrassCurve.toCharTwoJEqZeroNF_spec).

    • WeierstrassCurve.IsCharTwoJNeZeroNF is a type class which asserts that a WeierstrassCurve is of form Y¬≤ + XY = X¬≥ + a‚ÇÇX¬≤ + a‚ÇÜ. It is the normal form of characteristic = 2 and j ‚â† 0.
