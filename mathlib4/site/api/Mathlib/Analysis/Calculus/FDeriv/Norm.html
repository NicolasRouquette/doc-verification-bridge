    Differentiability of the norm in a real normed vector space #

    This file provides basic results about the differentiability of the norm in a real vector space. Most are of the following kind: if the norm has some differentiability property (DifferentiableAt, ContDiffAt, HasStrictFDerivAt, HasFDerivAt) at x, then so it has at t ‚Ä¢ x when t ‚â† 0.

    Main statements #

    • ContDiffAt.contDiffAt_norm_smul: If the norm is continuously differentiable up to order n at x, then so it is at t ‚Ä¢ x when t ‚â† 0.
    • differentiableAt_norm_smul: If t ‚â† 0, the norm is differentiable at x if and only if it is at t ‚Ä¢ x.
    • HasFDerivAt.hasFDerivAt_norm_smul: If the norm has a Fr√©chet derivative f at x and t ‚â† 0, then it has (SignType t) ‚Ä¢ f as a Fr√©chet derivative at t ¬∑ x.

    • DifferentiableAt.fderiv_norm_self: if the norm is differentiable at x, then fderiv ‚Ñù (‚Äñ¬∑‚Äñ) x x = ‚Äñx‚Äñ.
    • norm_fderiv_norm: if the norm is differentiable at x then the operator norm of its derivative is 1 (on a non-trivial space).
