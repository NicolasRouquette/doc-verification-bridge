    Real powers defined via the continuous functional calculus #

    This file defines real powers via the continuous functional calculus (CFC) and builds its API. This allows one to take real powers of matrices, operators, elements of a C‚ãÜ-algebra, etc. The square root is also defined via the non-unital CFC.

    Main declarations #

    • CFC.nnrpow: the ‚Ñù‚â•0 power function based on the non-unital CFC, i.e. cfc‚Çô NNReal.rpow composed with (‚Üë) : ‚Ñù‚â•0 ‚Üí ‚Ñù.
    • CFC.sqrt: the square root function based on the non-unital CFC, i.e. cfc‚Çô NNReal.sqrt
    • CFC.rpow: the real power function based on the unital CFC, i.e. cfc NNReal.rpow

    Implementation notes #

    We define two separate versions CFC.nnrpow and CFC.rpow due to what happens at 0. Since NNReal.rpow 0 0 = 1, this means that this function does not map zero to zero when the exponent is zero, and hence CFC.nnrpow a 0 = 0 whereas CFC.rpow a 0 = 1. Note that the non-unital version only makes sense for nonnegative exponents, and hence we define it such that the exponent is in ‚Ñù‚â•0.
