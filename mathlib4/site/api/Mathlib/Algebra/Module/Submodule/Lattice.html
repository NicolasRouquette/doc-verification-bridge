    The lattice structure on Submodules #

    This file defines the lattice structure on submodules, Submodule.CompleteLattice, with ‚ä• defined as {0} and ‚äì defined as intersection of the underlying carrier. If p and q are submodules of a module, p ‚â§ q means that p ‚äÜ q.

    Implementation notes #

    This structure should match the AddSubmonoid.CompleteLattice structure, and we should try to unify the APIs where possible.
