    Localized Module #

    Given a commutative semiring R, a multiplicative subset S ‚äÜ R and an R-module M, we can localize M by S. This gives us a Localization S-module.

    Main definitions #

    • LocalizedModule.r: the equivalence relation defining this localization, namely (m, s) ‚âà (m', s') if and only if there is some u : S such that u ‚Ä¢ s' ‚Ä¢ m = u ‚Ä¢ s ‚Ä¢ m'.
    • LocalizedModule M S: the localized module by S.
    • LocalizedModule.mk: the canonical map sending (m, s) : M √ó S ‚Ü¶ m/s : LocalizedModule M S
    • LocalizedModule.liftOn: any well-defined function f : M √ó S ‚Üí Œ± respecting r descents to a function LocalizedModule M S ‚Üí Œ±
    • LocalizedModule.liftOn‚ÇÇ: any well-defined function f : M √ó S ‚Üí M √ó S ‚Üí Œ± respecting r descents to a function LocalizedModule M S ‚Üí LocalizedModule M S
    • LocalizedModule.mk_add_mk: in the localized module mk m s + mk m' s' = mk (s' ‚Ä¢ m + s ‚Ä¢ m') (s * s')
    • LocalizedModule.mk_smul_mk : in the localized module, for any r : R, s t : S, m : M, we have mk r s ‚Ä¢ mk m t = mk (r ‚Ä¢ m) (s * t) where mk r s : Localization S is localized ring by S.
    • LocalizedModule.isModule : LocalizedModule M S is a Localization S-module.
