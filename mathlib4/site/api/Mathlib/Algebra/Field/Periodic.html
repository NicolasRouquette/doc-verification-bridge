    Periodic functions #

    This file proves facts about periodic and antiperiodic functions from and to a field.

    Main definitions #

    • Function.Periodic: A function f is periodic if ‚àÄ x, f (x + c) = f x. f is referred to as periodic with period c or c-periodic.

    • Function.Antiperiodic: A function f is antiperiodic if ‚àÄ x, f (x + c) = -f x. f is referred to as antiperiodic with antiperiod c or c-antiperiodic.

    Note that any c-antiperiodic function will necessarily also be 2 ‚Ä¢ c-periodic.
