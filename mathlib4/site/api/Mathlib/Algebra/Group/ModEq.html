    Equality modulo an element #

    This file defines equality modulo an element in an additive commutative monoid. In case of a group, a and b are congruent modulo p iff b - a ‚àà zmultiples p.

    In case of a monoid, the definition is a bit more complicated, and it is given with the use case of natural numbers in mind.

    Main definitions #

    • a ‚â° b [PMOD p]: a and b are congruent modulo p.

    See also #

    SModEq is a generalisation to arbitrary submodules.

    def AddCommGroup.ModEq {M : Type u_1} [AddCommMonoid M] (p a b : M) :
    Prop

    a ‚â° b [PMOD p] means that b is congruent to a modulo p.

    If a, b are elements of an additive group, then a ‚â° b [PMOD p] iff m ‚Ä¢ p = b - a for some m : ‚Ñ§, see modEq_iff_zsmul below. For additive commutative monoid, the definition is given by modEq_iff_nsmul.

    Equivalently (as shown in Algebra.Order.ToIntervalMod), b does not lie in the open interval (a, a + p) modulo p, or toIcoMod hp a disagrees with toIocMod hp a at b, or toIcoDiv hp a disagrees with toIocDiv hp a at b.
