    Associator in a ring #

    If R is a non-associative ring, then (x * y) * z - x * (y * z) is called the associator of ring elements x y z : R.

    The associator vanishes exactly when R is associative.

    We prove variants of this statement also for the AddMonoidHom bundled version of the associator, as well as the bundled version of mulLeft‚ÇÉ and mulRight‚ÇÉ, the multiplications (x * y) * z and x * (y * z).
