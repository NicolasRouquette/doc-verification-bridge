    Exactness of a pair #

    • For two maps f : M ‚Üí N and g : N ‚Üí P, with Zero P, Function.Exact f g says that Set.range f = Set.preimage g {0}

    • For two maps f : M ‚Üí N and g : N ‚Üí P, with One P, Function.MulExact f g says that Set.range f = Set.preimage g {1}

    • For additive maps f : M ‚Üí+ N and g : N ‚Üí+ P, Exact f g says that range f = ker g

    • For multiplicative maps f : M ‚Üí* N and g : N ‚Üí* P, MulExact f g says that range f = ker g

    • For linear maps f : M ‚Üí‚Çó[R] N and g : N ‚Üí‚Çó[R] P, Exact f g says that range f = ker g
