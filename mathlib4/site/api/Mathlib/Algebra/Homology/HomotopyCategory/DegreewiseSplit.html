    Degreewise split exact sequences of cochain complexes #

    The main result of this file is the lemma HomotopyCategory.distinguished_iff_iso_trianglehOfDegreewiseSplit which asserts that a triangle in HomotopyCategory C (ComplexShape.up ‚Ñ§) is distinguished iff it is isomorphic to the triangle attached to a degreewise split short exact sequence of cochain complexes.
