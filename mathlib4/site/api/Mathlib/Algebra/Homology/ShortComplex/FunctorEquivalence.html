    Short complexes in functor categories #

    In this file, it is shown that if J and C are two categories (such that C has zero morphisms), then there is an equivalence of categories ShortComplex.functorEquivalence J C : ShortComplex (J ‚•§ C) ‚âå J ‚•§ ShortComplex C.
