    Connecting a chain complex and a cochain complex #

    Given a chain complex K: ... ‚ü∂ K.X 2 ‚ü∂ K.X 1 ‚ü∂ K.X 0, a cochain complex L: L.X 0 ‚ü∂ L.X 1 ‚ü∂ L.X 2 ‚ü∂ ..., a morphism d‚ÇÄ : K.X 0 ‚ü∂ L.X 0 satisfying the identifies K.d 1 0 ‚â´ d‚ÇÄ = 0 and d‚ÇÄ ‚â´ L.d 0 1 = 0, we construct a cochain complex indexed by ‚Ñ§ of the form ... ‚ü∂ K.X 2 ‚ü∂ K.X 1 ‚ü∂ K.X 0 ‚ü∂ L.X 0 ‚ü∂ L.X 1 ‚ü∂ L.X 2 ‚ü∂ ..., where K.X 0 lies in degree -1 and L.X 0 in degree 0.

    Now say h : ConnectData K L.

    • CochainComplex.ConnectData.cochainComplex h : the induced ‚Ñ§-indexed complex ... ‚ü∂ K‚ÇÅ ‚ü∂ K‚ÇÄ ‚ü∂ L‚Å∞ ‚ü∂ L¬π ‚ü∂ ...
