    Homological complexes supported in a single degree #

    We define single V j c : V ‚•§ HomologicalComplex V c, which constructs complexes in V of shape c, supported in degree j.

    In ChainComplex.toSingle‚ÇÄEquiv we characterize chain maps to an ‚Ñï-indexed complex concentrated in degree 0; they are equivalent to { f : C.X 0 ‚ü∂ X // C.d 1 0 ‚â´ f = 0 }. (This is useful translating between a projective resolution and an augmented exact complex of projectives.)
