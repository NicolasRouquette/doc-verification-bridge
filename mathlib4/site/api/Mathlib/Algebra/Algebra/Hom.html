    Homomorphisms of R-algebras #

    This file defines bundled homomorphisms of R-algebras.

    Main definitions #

    • AlgHom R A B: the type of R-algebra morphisms from A to B.
    • Algebra.ofId R A : R ‚Üí‚Çê[R] A: the canonical map from R to A, as an AlgHom.

    Notation #

    • A ‚Üí‚Çê[R] B : R-algebra homomorphism from A to B.
