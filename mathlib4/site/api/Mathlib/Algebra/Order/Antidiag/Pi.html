    Antidiagonal of functions as finsets #

    This file provides the finset of functions summing to a specific value on a finset. Such finsets should be thought of as the "antidiagonals" in the space of functions.

    Precisely, for a commutative monoid Œº with antidiagonals (see Finset.HasAntidiagonal), Finset.piAntidiag s n is the finset of all functions f : Œπ ‚Üí Œº with support contained in s and such that the sum of its values equals n : Œº.

    We define it recursively on s using Finset.HasAntidiagonal.antidiagonal : Œº ‚Üí Finset (Œº √ó Œº). Technically, we non-canonically identify s with Fin n where n = s.card, recurse on n using that (Fin (n + 1) ‚Üí Œº) ‚âÉ (Fin n ‚Üí Œº) √ó Œº, and show the end result doesn't depend on our identification. See Finset.finAntidiag for the details.

    Main declarations #

    • Finset.piAntidiag s n: Finset of all functions f : Œπ ‚Üí Œº with support contained in s and such that the sum of its values equals n : Œº.
    • Finset.finAntidiagonal d n: Computationally efficient special case of Finset.piAntidiag when Œπ := Fin d.
