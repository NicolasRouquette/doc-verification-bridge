    Action of the polynomial ring on module induced by an algebra element. #

    Given an element a in an R-algebra A and an A-module M we define an R[X]-module Module.AEval R M a, which is a type synonym of M with the action of a polynomial f given by f ‚Ä¢ m = Polynomial.aeval a f ‚Ä¢ m. In particular X ‚Ä¢ m = a ‚Ä¢ m.

    Suppose a is an element of an R-algebra A and M is an A-module. Loosely speaking, Module.AEval R M a is the R[X]-module with elements m : M, where the action of a polynomial $f$ is given by $f ‚Ä¢ m = f(a) ‚Ä¢ m$.

    More precisely, Module.AEval R M a has elements Module.AEval.of R M a m for m : M, and the action of f is f ‚Ä¢ (of R M a m) = of R M a ((aeval a f) ‚Ä¢ m).
