    Presheaves of modules over a presheaf of rings. #

    Given a presheaf of rings R : C·µí·µñ ‚•§ RingCat, we define the category PresheafOfModules R. An object M : PresheafOfModules R consists of a family of modules M.obj X : ModuleCat (R.obj X) for all X : C·µí·µñ, together with the data, for all f : X ‚ü∂ Y, of a functorial linear map M.map f from M.obj X to the restriction of scalars of M.obj Y via R.map f.

    Future work #

    • Compare this to the definition as a presheaf of pairs (R, M) with specified first part.
    • Compare this to the definition as a module object of the presheaf of rings thought of as a monoid object.
    • Presheaves of modules over a presheaf of commutative rings form a monoidal category.
    • Pushforward and pullback.
