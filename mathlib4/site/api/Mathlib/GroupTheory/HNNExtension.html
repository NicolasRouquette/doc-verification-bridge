    HNN Extensions of Groups #

    This file defines the HNN extension of a group G, HNNExtension G A B œÜ. Given a group G, subgroups A and B and an isomorphism œÜ of A and B, we adjoin a letter t to G, such that for any a ‚àà A, the conjugate of of a by t is of (œÜ a), where of is the canonical map from G into the HNNExtension. This construction is named after Graham Higman, Bernhard Neumann and Hanna Neumann.

    Main definitions #

    • HNNExtension G A B œÜ : The HNN Extension of a group G, where A and B are subgroups and œÜ is an isomorphism between A and B.
    • HNNExtension.of : The canonical embedding of G into HNNExtension G A B œÜ.
    • HNNExtension.t : The stable letter of the HNN extension.
    • HNNExtension.lift : Define a function HNNExtension G A B œÜ ‚Üí* H, by defining it on G and t
    • HNNExtension.of_injective : The canonical embedding G ‚Üí* HNNExtension G A B œÜ is injective.
    • HNNExtension.ReducedWord.toList_eq_nil_of_mem_of_range : Britton's Lemma. If an element of G is represented by a reduced word, then this reduced word does not contain t.
