    Properties of fixedPoints and fixedBy #

    This module contains some useful properties of MulAction.fixedPoints and MulAction.fixedBy that don't directly belong to Mathlib/GroupTheory/GroupAction/Basic.lean, as well as their interaction with MulActionHom.

    Main theorems #

    • MulAction.fixedBy_mul: fixedBy Œ± (g * h) ‚äÜ fixedBy Œ± g ‚à™ fixedBy Œ± h
    • MulAction.fixedBy_conj and MulAction.smul_fixedBy: the pointwise group action of h on fixedBy Œ± g is equal to the fixedBy set of the conjugation of h with g (fixedBy Œ± (h * g * h‚Åª¬π)).
    • MulAction.set_mem_fixedBy_of_movedBy_subset shows that if a set s is a superset of (fixedBy Œ± g)·∂ú, then the group action of g cannot send elements of s outside of s. This is expressed as s ‚àà fixedBy (Set Œ±) g, and MulAction.set_mem_fixedBy_iff allows one to convert the relationship back to g ‚Ä¢ x ‚àà s ‚Üî x ‚àà s.
    • MulAction.not_commute_of_disjoint_smul_movedBy allows one to prove that g and h do not commute from the disjointness of the (fixedBy Œ± g)·∂ú set and h ‚Ä¢ (fixedBy Œ± g)·∂ú, which is a property used in the proof of Rubin's theorem.

    The theorems above are also available for AddAction.

    Pointwise group action and fixedBy (Set Œ±) g #

    Since fixedBy Œ± g = { x | g ‚Ä¢ x = x } by definition, properties about the pointwise action of a set s : Set Œ± can be expressed using fixedBy (Set Œ±) g. To properly use theorems using fixedBy (Set Œ±) g, you should open Pointwise in your file.

    s ‚àà fixedBy (Set Œ±) g means that g ‚Ä¢ s = s, which is equivalent to say that ‚àÄ x, g ‚Ä¢ x ‚àà s ‚Üî x ‚àà s (the translation can be done using MulAction.set_mem_fixedBy_iff).

    s ‚àà fixedBy (Set Œ±) g is a weaker statement than s ‚äÜ fixedBy Œ± g: the latter requires that all points in s are fixed by g, whereas the former only requires that g ‚Ä¢ x ‚àà s.

    Pointwise image of the fixedBy set by a commuting group element #

    If two group elements g and h commute, then g fixes h ‚Ä¢ x (resp. h +·µ• x) if and only if g fixes x.

    This is equivalent to say that if Commute g h, then fixedBy Œ± g ‚àà fixedBy (Set Œ±) h and (fixedBy Œ± g)·∂ú ‚àà fixedBy (Set Œ±) h.
