    Von Neumann ordinals #

    This file works towards the development of von Neumann ordinals, i.e. transitive sets, well-ordered under âˆˆ.

    Definitions #

    • ZFSet.IsTransitive means that every element of a set is a subset.
    • ZFSet.IsOrdinal means that the set is transitive and well-ordered under âˆˆ. We show multiple equivalences to this definition.
    • Ordinal.toZFSet converts Lean's type-theoretic ordinals into ZFC ordinals. We prove that these two notions are order-isomorphic.
