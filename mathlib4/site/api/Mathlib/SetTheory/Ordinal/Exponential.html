    Ordinal exponential #

    In this file we define the power function and the logarithm function on ordinals. The two are related by the lemma Ordinal.opow_le_iff_le_log : b ^ c ‚â§ x ‚Üî c ‚â§ log b x for nontrivial inputs b, c.
