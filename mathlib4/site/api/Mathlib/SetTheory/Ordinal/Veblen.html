    Veblen hierarchy #

    We define the two-arguments Veblen function, which satisfies veblen 0 a = œâ ^ a and that for o ‚â† 0, veblen o enumerates the common fixed points of veblen o' for o' < o.

    We use this to define two important functions on ordinals: the epsilon function Œµ_ o = veblen 1 o, and the gamma function Œì_ o enumerating the fixed points of veblen ¬∑ 0.

    Main definitions #

    • veblenWith: The Veblen hierarchy with a specified initial function.
    • veblen: The Veblen hierarchy starting with œâ ^ ¬∑.

    Notation #

    The following notation is scoped to the Ordinal namespace.

    • Œµ_ o is notation for veblen 1 o. Œµ‚ÇÄ is notation for Œµ_ 0.
    • Œì_ o is notation for gamma o. Œì‚ÇÄ is notation for Œì_ 0.
