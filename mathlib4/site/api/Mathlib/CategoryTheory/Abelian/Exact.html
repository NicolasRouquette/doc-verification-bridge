    Exact sequences in abelian categories #

    In an abelian category, we get several interesting results related to exactness which are not true in more general settings.

    Main results #

    • A short complex S is exact iff imageSubobject S.f = kernelSubobject S.g.
    • If (f, g) is exact, then image.Œπ f has the universal property of the kernel of g.
    • f is a monomorphism iff kernel.Œπ f = 0 iff Exact 0 f, and f is an epimorphism iff cokernel.œÄ f = 0 iff Exact f 0.
    • A faithful functor between abelian categories that preserves zero morphisms reflects exact sequences.
    • X ‚ü∂ Y ‚ü∂ Z ‚ü∂ 0 is exact if and only if the second map is a cokernel of the first, and 0 ‚ü∂ X ‚ü∂ Y ‚ü∂ Z is exact if and only if the first map is a kernel of the second.
    • A functor F such that for all S, we have S.Exact ‚Üí (S.map F).Exact preserves both finite limits and colimits.
