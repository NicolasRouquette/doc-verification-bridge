    Adjunctions with a parameter #

    Given bifunctors F : C‚ÇÅ ‚•§ C‚ÇÇ ‚•§ C‚ÇÉ and G : C‚ÇÅ·µí·µñ ‚•§ C‚ÇÉ ‚•§ C‚ÇÇ, this file introduces the notation F ‚ä£‚ÇÇ G for the adjunctions with a parameter (in C‚ÇÅ) between F and G.

    (See MonoidalClosed.internalHomAdjunction‚ÇÇ in the file CategoryTheory.Closed.Monoidal for an example of such an adjunction.)

    Note: this notion is weaker than the notion of "adjunction of two variables" which appears in the mathematical literature.
