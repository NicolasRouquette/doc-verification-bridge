    Colimit of representables #

    In this file, We show that every presheaf of types on a category C (with Category.{v‚ÇÅ} C) is a colimit of representables. This result is also known as the density theorem, the co-Yoneda lemma and the Ninja Yoneda lemma. Three formulations are given:

    • colimitOfRepresentable uses the category of elements of a functor to types;
    • isColimitTautologicalCocone uses the category of costructured arrows for yoneda : C ‚•§ C·µí·µñ ‚•§ Type v‚ÇÅ;
    • isColimitTautologicalCocone' uses the category of costructured arrows for uliftYoneda : C ‚•§ C·µí·µñ ‚•§ Type max w v‚ÇÅ, when the presheaf has values in Type (max w v‚ÇÅ);

    In this file, we also study the left Kan extensions of functors A : C ‚•§ ‚Ñ∞ along the Yoneda embedding uliftYoneda : C ‚•§ C·µí·µñ ‚•§ Type max w v‚ÇÅ v‚ÇÇ (when Category.{v‚ÇÇ} ‚Ñ∞ and w is an auxiliary universe).     In the lemma isLeftKanExtension_along_uliftYoneda_iff, we show that if L : (C·µí·µñ ‚•§ Type max w v‚ÇÅ v‚ÇÇ) ‚•§ ‚Ñ∞ and Œ± : A ‚ü∂ uliftYoneda ‚ãô L, then Œ± makes L the left Kan extension of L along yoneda if and only if Œ± is an isomorphism (i.e. L extends A) and L preserves colimits. uniqueExtensionAlongULiftYoneda shows uliftYoneda.leftKanExtension A is unique amongst functors preserving colimits with this property, establishing the presheaf category as the free cocompletion of a category.

    Given a functor F : C ‚•§ D, we also show construct an isomorphism compULiftYonedaIsoULiftYonedaCompLan : F ‚ãô uliftYoneda ‚âÖ uliftYoneda ‚ãô F.op.lan, and show that it makes F.op.lan a left Kan extension of F ‚ãô uliftYoneda.
