    Connected limits #

    A connected limit is a limit whose shape is a connected category.

    We show that constant functors from a connected category have a limit and a colimit. From this we deduce that a cocone c over a connected diagram is a colimit cocone if and only if colimMap c.Œπ is an isomorphism (where c.Œπ : F ‚ü∂ const c.pt is the natural transformation that defines the cocone).

    We give examples of connected categories, and prove that the functor given by (X √ó -) preserves any connected limit. That is, any limit of shape J where J is a connected category is preserved by the functor (X √ó -).
