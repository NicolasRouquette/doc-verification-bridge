    Relation between mono/epi and pullback/pushout squares #

    In this file, monomorphisms and epimorphisms are characterized in terms of pullback and pushout squares. For example, we obtain mono_iff_isPullback which asserts that a morphism f : X ‚ü∂ Y is a monomorphism iff the obvious square

    X ‚ü∂ X
|   |
v   v
X ‚ü∂ Y

    is a pullback square.
