    Coproducts in Type #

    If F : J ‚Üí Type max v u (with J : Type v), we show that the coproduct of F exists in Type max v u and identifies to the sigma type Œ£ j, F j. Similarly, the binary coproduct of two types X and Y identifies to X ‚äï Y, and the initial object of Type u if PEmpty.
