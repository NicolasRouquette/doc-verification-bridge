    Pseudofunctors from locally discrete bicategories #

    This file provides various ways of constructing pseudofunctors from locally discrete bicategories.

    Firstly, we define the constructors pseudofunctorOfIsLocallyDiscrete and oplaxFunctorOfIsLocallyDiscrete for defining pseudofunctors and oplax functors from a locally discrete bicategory. In this situation, we do not need to care about the field map‚ÇÇ, because all the 2-morphisms in B are identities.

    We also define a specialized constructor LocallyDiscrete.mkPseudofunctor when the source bicategory is of the form B := LocallyDiscrete B‚ÇÄ for a category B‚ÇÄ.

    We also prove that a functor F : I ‚•§ B with B a strict bicategory can be promoted to a pseudofunctor (or oplax functor) (Functor.toPseudofunctor) with domain LocallyDiscrete I.
