    Kleisli category on a (co)monad #

    This file defines the Kleisli category on a monad (T, Œ∑_ T, Œº_ T) as well as the co-Kleisli category on a comonad (U, Œµ_ U, Œ¥_ U). It also defines the Kleisli adjunction which gives rise to the monad (T, Œ∑_ T, Œº_ T) as well as the co-Kleisli adjunction which gives rise to the comonad (U, Œµ_ U, Œ¥_ U).

    References #

    • [Riehl, Category theory in context, Definition 5.2.9][riehl2017]
