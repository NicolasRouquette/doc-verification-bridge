    The category of graded objects #

    For any type Œ≤, a Œ≤-graded object over some category C is just a function Œ≤ ‚Üí C into the objects of C. We put the "pointwise" category structure on these, as the non-dependent specialization of CategoryTheory.Pi.

    We describe the comap functors obtained by precomposing with functions Œ≤ ‚Üí Œ≥.

    As a consequence a fixed element (e.g. 1) in an additive group Œ≤ provides a shift functor on Œ≤-graded objects

    When C has coproducts we construct the total functor GradedObject Œ≤ C ‚•§ C, show that it is faithful, and deduce that when C is concrete so is GradedObject Œ≤ C.

    A covariant functoriality of GradedObject Œ≤ C with respect to the index set Œ≤ is also introduced: if p : I ‚Üí J is a map such that C has coproducts indexed by p ‚Åª¬π' {j}, we have a functor map : GradedObject I C ‚•§ GradedObject J C.
