    Natural isomorphisms #

    For the most part, natural isomorphisms are just another sort of isomorphism.

    We provide some special support for extracting components:

    • if Œ± : F ‚âÖ G, then Œ±.app X : F.obj X ‚âÖ G.obj X, and building natural isomorphisms from components:
    NatIso.ofComponents
  (app : ‚àÄ X : C, F.obj X ‚âÖ G.obj X)
  (naturality : ‚àÄ {X Y : C} (f : X ‚ü∂ Y), F.map f ‚â´ (app Y).hom = (app X).hom ‚â´ G.map f) :
F ‚âÖ G

    only needing to check naturality in one direction.

    Implementation #

    Note that NatIso is a namespace without a corresponding definition; we put some declarations that are specifically about natural isomorphisms in the Iso namespace so that they are available using dot notation.

      Unfortunately we need a separate set of cancellation lemmas for components of natural isomorphisms, because the simp normal form is Œ±.hom.app X, rather than Œ±.app.hom X.

      (With the latter, the morphism would be visibly part of an isomorphism, so general lemmas about isomorphisms would apply.)

      In the future, we should consider a redesign that changes this simp normal form, but for now it breaks too many proofs.
