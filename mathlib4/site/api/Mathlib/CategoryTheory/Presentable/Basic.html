    Presentable objects #

    A functor F : C ‚•§ D is Œ∫-accessible (Functor.IsCardinalAccessible) if it commutes with colimits of shape J where J is any Œ∫-filtered category (that is essentially small relative to the universe w such that Œ∫ : Cardinal.{w}.). We also introduce another typeclass Functor.IsAccessible saying that there exists a regular cardinal Œ∫ such that Functor.IsCardinalAccessible.

    An object X of a category is Œ∫-presentable (IsCardinalPresentable) if the functor Hom(X, _) (i.e. coyoneda.obj (op X)) is Œ∫-accessible. Similarly as for accessible functors, we define a type class IsAccessible.
