    Œ∫-filtered category #

    If Œ∫ is a regular cardinal, we introduce the notion of Œ∫-filtered category J: it means that any functor A ‚•§ J from a small category such that Arrow A is of cardinality < Œ∫ admits a cocone. This generalizes the notion of filtered category. Indeed, we obtain the equivalence IsCardinalFiltered J ‚Ñµ‚ÇÄ ‚Üî IsFiltered J. The API is mostly parallel to that of filtered categories.

    A preordered type J is a Œ∫-filtered category (i.e. Œ∫-directed set) if any subset of J of cardinality < Œ∫ has an upper bound.
