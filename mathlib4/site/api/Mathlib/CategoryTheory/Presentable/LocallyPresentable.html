    Locally presentable and accessible categories #

    In this file, we define the notion of locally presentable and accessible categories. We first define these notions for a category C relative to a fixed regular cardinal Œ∫ (typeclasses IsCardinalLocallyPresentable C Œ∫ and IsCardinalAccessibleCategory C Œ∫). The existence of such a regular cardinal Œ∫ is asserted in the typeclasses IsLocallyPresentable and IsAccessibleCategory. We show that in a locally presentable or accessible category, any object is presentable.
