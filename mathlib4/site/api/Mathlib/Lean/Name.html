          def Lean.Name.willRoundTrip (n : Name) :
          Bool

          Determines if the pretty-printed version of the given name would parse as an ident with an underlying name (via getId) equal to the original name. The pretty-printer usually escapes unparsable components of a name with Â«Â», but makes exceptions for various names with special meaning, meaning that the result does not round trip. We therefore re-check those conditions here.

          This function is intended to be "safe" in that if it returns true, the name will definitely round trip. (The converse is not guaranteed.) Any deviation from this behavior is a bug which should be fixed.
