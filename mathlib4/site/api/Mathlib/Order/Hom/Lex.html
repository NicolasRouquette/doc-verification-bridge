    Lexicographic order and order isomorphisms #

    Main declarations #

    • OrderIso.sumLexIioIci and OrderIso.sumLexIicIoi: if Œ± is a linear order and x : Œ±, then Œ± is order isomorphic to both Iio x ‚äï‚Çó Ici x and Iic x ‚äï‚Çó Ioi x.
    • Prod.Lex.prodUnique and Prod.Lex.uniqueProd: Œ± √ó‚Çó Œ≤ is order isomorphic to one side if the other side is Unique.
