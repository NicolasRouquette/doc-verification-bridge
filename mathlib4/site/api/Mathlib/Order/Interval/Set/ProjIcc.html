    Projection of a line onto a closed interval #

    Given a linearly ordered type Œ±, in this file we define

    • Set.projIci (a : Œ±) to be the map Œ± ‚Üí [a, ‚àû) sending (-‚àû, a] to a, and each point x ‚àà [a, ‚àû) to itself;
    • Set.projIic (b : Œ±) to be the map Œ± ‚Üí (-‚àû, b[ sending [b, ‚àû) to b, and each point x ‚àà (-‚àû, b] to itself;
    • Set.projIcc (a b : Œ±) (h : a ‚â§ b) to be the map Œ± ‚Üí [a, b] sending (-‚àû, a] to a, [b, ‚àû) to b, and each point x ‚àà [a, b] to itself;
    • Set.IccExtend {a b : Œ±} (h : a ‚â§ b) (f : Icc a b ‚Üí Œ≤) to be the extension of f to Œ± defined as f ‚àò projIcc a b h.
    • Set.IciExtend {a : Œ±} (f : Ici a ‚Üí Œ≤) to be the extension of f to Œ± defined as f ‚àò projIci a.
    • Set.IicExtend {b : Œ±} (f : Iic b ‚Üí Œ≤) to be the extension of f to Œ± defined as f ‚àò projIic b.

    We also prove some trivial properties of these maps.
