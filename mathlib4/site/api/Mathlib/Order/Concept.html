    Formal concept analysis #

    This file defines concept lattices. A concept of a relation r : Œ± ‚Üí Œ≤ ‚Üí Prop is a pair of sets s : Set Œ± and t : Set Œ≤ such that s is the set of all a : Œ± that are related to all elements of t, and t is the set of all b : Œ≤ that are related to all elements of s.

    Ordering the concepts of a relation r by inclusion on the first component gives rise to a concept lattice. Every concept lattice is complete and in fact every complete lattice arises as the concept lattice of its ‚â§.

    Implementation notes #

    Concept lattices are usually defined from a context, that is the triple (Œ±, Œ≤, r), but the type of r determines Œ± and Œ≤ already, so we do not define contexts as a separate object.
