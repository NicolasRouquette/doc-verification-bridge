    structure Sublattice (Œ± : Type u_2) [Lattice Œ±] :
    Type u_2

    A sublattice of a lattice is a set containing the suprema and infima of any of its elements.

    • carrier : Set Œ±

      The underlying set of a sublattice. Do not use directly. Instead, use the coercion Sublattice Œ± ‚Üí Set Œ±, which Lean should automatically insert for you in most cases.

    • supClosed' : SupClosed self.carrier
    • infClosed' : InfClosed self.carrier
