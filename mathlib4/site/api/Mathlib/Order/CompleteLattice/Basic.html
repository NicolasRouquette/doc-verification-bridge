    Theory of complete lattices #

    This file contains basic results on complete lattices.

    Naming conventions #

    In lemma names,

    • sSup is called sSup
    • sInf is called sInf
    • ‚®Ü i, s i is called iSup
    • ‚®Ö i, s i is called iInf
    • ‚®Ü i j, s i j is called iSup‚ÇÇ. This is an iSup inside an iSup.
    • ‚®Ö i j, s i j is called iInf‚ÇÇ. This is an iInf inside an iInf.
    • ‚®Ü i ‚àà s, t i is called biSup for "bounded iSup". This is the special case of iSup‚ÇÇ where j : i ‚àà s.
    • ‚®Ö i ‚àà s, t i is called biInf for "bounded iInf". This is the special case of iInf‚ÇÇ where j : i ‚àà s.

    Notation #

    • ‚®Ü i, f i : iSup f, the supremum of the range of f;
    • ‚®Ö i, f i : iInf f, the infimum of the range of f.
