    Antichains #

    This file defines antichains. An antichain is a set where any two distinct elements are not related. If the relation is (‚â§), this corresponds to incomparability and usual order antichains. If the relation is G.Adj for G : SimpleGraph Œ±, this corresponds to independent sets of G.

    Definitions #

    • IsAntichain r s: Any two elements of s : Set Œ± are unrelated by r : Œ± ‚Üí Œ± ‚Üí Prop.
    • IsStrongAntichain r s: Any two elements of s : Set Œ± are not related by r : Œ± ‚Üí Œ± ‚Üí Prop to a common element.
    • IsMaxAntichain r s: An antichain such that no antichain strictly including s exists.
