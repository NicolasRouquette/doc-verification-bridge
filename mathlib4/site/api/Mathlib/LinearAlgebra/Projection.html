    Projection to a subspace #

    In this file we define

    • Submodule.linearProjOfIsCompl (p q : Submodule R E) (h : IsCompl p q): the projection of a module E to a submodule p along its complement q; it is the unique linear map f : E ‚Üí p such that f x = x for x ‚àà p and f x = 0 for x ‚àà q.
    • Submodule.isComplEquivProj p: equivalence between submodules q such that IsCompl p q and projections f : E ‚Üí p, ‚àÄ x ‚àà p, f x = x.

    We also provide some lemmas justifying correctness of our definitions.
