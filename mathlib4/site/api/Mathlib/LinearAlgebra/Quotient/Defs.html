    Quotients by submodules #

    • If p is a submodule of M, M ‚ß∏ p is the quotient of M with respect to p: that is, elements of M are identified if their difference is in p. This is itself a module.

    Main definitions #

    • Submodule.Quotient.mk: a function sending an element of M to M ‚ß∏ p
    • Submodule.Quotient.module: M ‚ß∏ p is a module
    • Submodule.Quotient.mkQ: a linear map sending an element of M to M ‚ß∏ p
    • Submodule.quotEquivOfEq: if p and p' are equal, their quotients are equivalent

    Setoid M

    The equivalence relation associated to a submodule p, defined by x ‚âà y iff -x + y ‚àà p.

    Note this is equivalent to y - x ‚àà p, but defined this way to be defeq to the AddSubgroup version, where commutativity can't be assumed.
