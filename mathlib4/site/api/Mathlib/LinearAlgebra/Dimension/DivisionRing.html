    Dimension of vector spaces #

    In this file we provide results about Module.rank and Module.finrank of vector spaces over division rings.

    Main statements #

    For vector spaces (i.e. modules over a field), we have

    • rank_quotient_add_rank_of_divisionRing: if V‚ÇÅ is a submodule of V, then Module.rank (V/V‚ÇÅ) + Module.rank V‚ÇÅ = Module.rank V.
    • rank_range_add_rank_ker: the rank-nullity theorem.

    See also Mathlib/LinearAlgebra/Dimension/ErdosKaplansky.lean for the Erd≈ës-Kaplansky theorem.
