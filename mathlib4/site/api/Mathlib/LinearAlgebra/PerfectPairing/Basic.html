    Perfect pairings #

    This file defines perfect pairings of modules.

    A perfect pairing of two (left) modules may be defined either as:

    1. A bilinear map M Ã— N â†’ R such that the induced maps M â†’ Dual R N and N â†’ Dual R M are both bijective. It follows from this that both M and N are reflexive modules.
    2. A linear equivalence N â‰ƒ Dual R M for which M is reflexive. (It then follows that N is reflexive.)

    In this file we provide a definition IsPerfPair corresponding to 1 above, together with logic to connect 1 and 2.
