    Tensor product and products #

    In this file we examine the behaviour of the tensor product with arbitrary and finite products.

    Let S be an R-algebra, N an S-module, Œπ an index type and M·µ¢ a family of R-modules. We then have a natural map

    TensorProduct.piRightHom: N ‚äó[R] (‚àÄ i, M i) ‚Üí‚Çó[S] ‚àÄ i, N ‚äó[R] M i

    In general, this is not an isomorphism, but if Œπ is finite, then it is and it is packaged as TensorProduct.piRight. Also a special case for when M·µ¢ = R is given.

    Notes #

    See Mathlib/LinearAlgebra/TensorProduct/Prod.lean for binary products.
