    Tensor power of a semimodule over a commutative semiring #

    We define the nth tensor power of M as the n-ary tensor product indexed by Fin n of M, ‚®Ç[R] (i : Fin n), M. This is a special case of PiTensorProduct.

    This file introduces the notation ‚®Ç[R]^n M for TensorPower R n M, which in turn is an abbreviation for ‚®Ç[R] i : Fin n, M.

    Main definitions: #

    • TensorPower.gsemiring: the tensor powers form a graded semiring.
    • TensorPower.galgebra: the tensor powers form a graded algebra.

    Implementation notes #

    In this file we use ‚Çú1 and ‚Çú* as local notation for the graded multiplicative structure on tensor powers. Elsewhere, using 1 and * on GradedMonoid should be preferred.
