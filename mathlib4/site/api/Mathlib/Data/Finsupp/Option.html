    Operations on Finsupps with an Option domain #

    Similar to how Finsupp.cons and Finsupp.tail construct an object of type Fin (n + 1) ‚Üí‚ÇÄ M from a map Fin n ‚Üí‚ÇÄ M and vice versa, we define Finsupp.optionElim and Finsupp.some to construct Option Œ± ‚Üí‚ÇÄ M from a map Œ± ‚Üí‚ÇÄ M, and vice versa.

    As functions, these behave as Option.elim', and as an application of some hence the names.

    We prove a variety of API lemmas, see Mathlib/Data/Finsupp/Fin.lean for comparison.

    Main declarations #

    • Finsupp.some: restrict a finitely supported function on Option Œ± to a finitely supported function on Œ±.
    • Finsupp.optionElim: extend a finitely supported function on Œ± to a finitely supported function on Option Œ±, provided a default value for none.
