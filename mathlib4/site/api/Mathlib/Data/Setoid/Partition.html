    Equivalence relations: partitions #

    This file comprises properties of equivalence relations viewed as partitions. There are two implementations of partitions here:

    • A collection c : Set (Set Œ±) of sets is a partition of Œ± if ‚àÖ ‚àâ c and each element a : Œ± belongs to a unique set b ‚àà c. This is expressed as IsPartition c
    • An indexed partition is a map s : Œπ ‚Üí Œ± whose image is a partition. This is expressed as IndexedPartition s.

    Of course both implementations are related to Quotient and Setoid.

    Setoid.isPartition.partition and Finpartition.isPartition_parts furnish a link between Setoid.IsPartition and Finpartition.

                  Constructive information associated with a partition of a type Œ± indexed by another type Œπ, s : Œπ ‚Üí Set Œ±.
