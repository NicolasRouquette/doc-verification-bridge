    Intervals of finsets as finsets #

    This file provides the LocallyFiniteOrder instance for Finset Î± and calculates the cardinality of finite intervals of finsets.

    If s t : Finset Î±, then Finset.Icc s t is the finset of finsets which include s and are included in t. For example, Finset.Icc {0, 1} {0, 1, 2, 3} = {{0, 1}, {0, 1, 2}, {0, 1, 3}, {0, 1, 2, 3}} and Finset.Icc {0, 1, 2} {0, 1, 3} = {}.
