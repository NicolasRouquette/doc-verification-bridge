    Sorting a finite type #

    This file provides two equivalences for linearly ordered fintypes:

    • monoEquivOfFin: Order isomorphism between Œ± and Fin (card Œ±).
    • finSumEquivOfFinset: Equivalence between Œ± and Fin m ‚äï Fin n where m and n are respectively the cardinalities of some Finset Œ± and its complement.
