    Links between an integral and its "improper" version #

    In its current state, mathlib only knows how to talk about definite ("proper") integrals, in the sense that it treats integrals over [x, +‚àû) the same as it treats integrals over [y, z]. For example, the integral over [1, +‚àû) is not defined to be the limit of the integral over [1, x] as x tends to +‚àû, which is known as an improper integral.

    Indeed, the "proper" definition is stronger than the "improper" one. The usual counterexample is x ‚Ü¶ sin(x)/x, which has an improper integral over [1, +‚àû) but no definite integral.

    Although definite integrals have better properties, they are hardly usable when it comes to computing integrals on unbounded sets, which is much easier using limits. Thus, in this file, we prove various ways of studying the proper integral by studying the improper one.

    Definitions #

    The main definition of this file is MeasureTheory.AECover. It is a rather technical definition whose sole purpose is generalizing and factoring proofs. Given an index type Œπ, a countably generated filter l over Œπ, and an Œπ-indexed family œÜ of subsets of a measurable space Œ± equipped with a measure Œº, one should think of a hypothesis hœÜ : MeasureTheory.AECover Œº l œÜ as a sufficient condition for being able to interpret ‚à´ x, f x ‚àÇŒº (if it exists) as the limit of ‚à´ x in œÜ i, f x ‚àÇŒº as i tends to l.

    When using this definition with a measure restricted to a set s, which happens fairly often, one should not try too hard to use a MeasureTheory.AECover of subsets of s, as it often makes proofs more complicated than necessary.
