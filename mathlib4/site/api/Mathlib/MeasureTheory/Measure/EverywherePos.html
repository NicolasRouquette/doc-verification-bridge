    Everywhere positive sets in measure spaces #

    A set s in a topological space with a measure Œº is everywhere positive (also called self-supporting) if any neighborhood n of any point of s satisfies Œº (s ‚à© n) > 0.

    Main definitions and results #

    • Œº.IsEverywherePos s registers that, for any point in s, all its neighborhoods have positive measure inside s.
    • Œº.everywherePosSubset s is the subset of s made of those points all of whose neighborhoods have positive measure inside s.
    • everywherePosSubset_ae_eq shows that s and Œº.everywherePosSubset s coincide almost everywhere if Œº is inner regular and s is measurable.
    • isEverywherePos_everywherePosSubset shows that Œº.everywherePosSubset s satisfies the property Œº.IsEverywherePos if Œº is inner regular and s is measurable.

    The latter two statements have also versions when Œº is inner regular for finite measure sets, assuming additionally that s has finite measure.

    • IsEverywherePos.IsGŒ¥ proves that an everywhere positive compact closed set is a GŒ¥ set, in a topological group with a left-invariant measure. This is a nontrivial statement, used crucially in the study of the uniqueness of Haar measures.
    • innerRegularWRT_preimage_one_hasCompactSupport_measure_ne_top: for a Haar measure, any finite measure set can be approximated from inside by level sets of continuous compactly supported functions. This property is also known as completion-regularity of Haar measures.
