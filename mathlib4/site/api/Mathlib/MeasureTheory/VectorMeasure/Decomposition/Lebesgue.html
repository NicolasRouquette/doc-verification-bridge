    Lebesgue decomposition #

    This file proves the Lebesgue decomposition theorem for signed measures. The Lebesgue decomposition theorem states that, given two œÉ-finite measures Œº and ŒΩ, there exists a œÉ-finite measure Œæ and a measurable function f such that Œº = Œæ + fŒΩ and Œæ is mutually singular with respect to ŒΩ.

    Main definitions #

    • MeasureTheory.SignedMeasure.HaveLebesgueDecomposition : A signed measure s is said to have Lebesgue decomposition with respect to a measure Œº if both the positive part and negative part of s have Lebesgue decomposition with respect to Œº.
    • MeasureTheory.SignedMeasure.singularPart : The singular part between a signed measure s and a measure Œº is simply the singular part of the positive part of s with respect to Œº minus the singular part of the negative part of s with respect to Œº.
    • MeasureTheory.SignedMeasure.rnDeriv : The Radon-Nikodym derivative of a signed measure s with respect to a measure Œº is the Radon-Nikodym derivative of the positive part of s with respect to Œº minus the Radon-Nikodym derivative of the negative part of s with respect to Œº.

    Main results #

    • MeasureTheory.SignedMeasure.singularPart_add_withDensity_rnDeriv_eq : the Lebesgue decomposition theorem between a signed measure and a œÉ-finite positive measure.
