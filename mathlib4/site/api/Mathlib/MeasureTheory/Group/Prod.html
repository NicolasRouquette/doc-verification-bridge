    Measure theory in the product of groups #

    In this file we show properties about measure theory in products of measurable groups and properties of iterated integrals in measurable groups.

    These lemmas show the uniqueness of left invariant measures on measurable groups, up to scaling. In this file we follow the proof and refer to the book Measure Theory by Paul Halmos.

    The idea of the proof is to use the translation invariance of measures to prove Œº(t) = c * Œº(s) for two sets s and t, where c is a constant that does not depend on Œº. Let e and f be the characteristic functions of s and t. Assume that Œº and ŒΩ are left-invariant measures. Then the map (x, y) ‚Ü¶ (y * x, x‚Åª¬π) preserves the measure Œº √ó ŒΩ, which means that

      ‚à´ x, ‚à´ y, h x y ‚àÇŒΩ ‚àÇŒº = ‚à´ x, ‚à´ y, h (y * x) x‚Åª¬π ‚àÇŒΩ ‚àÇŒº

    If we apply this to h x y := e x * f y‚Åª¬π / ŒΩ ((fun h ‚Ü¶ h * y‚Åª¬π) ‚Åª¬π' s), we can rewrite the RHS to Œº(t), and the LHS to c * Œº(s), where c = c(ŒΩ) does not depend on Œº. Applying this to Œº and to ŒΩ gives Œº (t) / Œº (s) = ŒΩ (t) / ŒΩ (s), which is the uniqueness up to scalar multiplication.

    The proof in [Halmos] seems to contain an omission in ¬ß60 Th. A, see MeasureTheory.measure_lintegral_div_measure.

    Note that this theory only applies in measurable groups, i.e., when multiplication and inversion are measurable. This is not the case in general in locally compact groups, or even in compact groups, when the topology is not second-countable. For arguments along the same line, but using continuous functions instead of measurable sets and working in the general locally compact setting, see the file Mathlib/MeasureTheory/Measure/Haar/Unique.lean.
