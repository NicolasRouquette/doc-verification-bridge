    Relative rank of subfields and intermediate fields #

    This file contains basics about the relative rank of subfields and intermediate fields.

    Main definitions #

    • Subfield.relrank A B, IntermediateField.relrank A B: defined to be [B : A ‚äì B] as a Cardinal. In particular, when A ‚â§ B it is [B : A], the degree of the field extension B / A. This is similar to Subgroup.relIndex but it is Cardinal valued.

    • Subfield.relfinrank A B, IntermediateField.relfinrank A B: the Nat version of Subfield.relrank A B and IntermediateField.relrank A B, respectively. If B / A ‚äì B is an infinite extension, then it is zero. This is similar to Subgroup.relIndex.
