    Ordering on topologies and (co)induced topologies #

    Topologies on a fixed type Œ± are ordered, by reverse inclusion. That is, for topologies t‚ÇÅ and t‚ÇÇ on Œ±, we write t‚ÇÅ ‚â§ t‚ÇÇ if every set open in t‚ÇÇ is also open in t‚ÇÅ. (One also calls t‚ÇÅ finer than t‚ÇÇ, and t‚ÇÇ coarser than t‚ÇÅ.)

    Any function f : Œ± ‚Üí Œ≤ induces

    • TopologicalSpace.induced f : TopologicalSpace Œ≤ ‚Üí TopologicalSpace Œ±;
    • TopologicalSpace.coinduced f : TopologicalSpace Œ± ‚Üí TopologicalSpace Œ≤.

    Continuity, the ordering on topologies and (co)induced topologies are related as follows:

    • The identity map (Œ±, t‚ÇÅ) ‚Üí (Œ±, t‚ÇÇ) is continuous iff t‚ÇÅ ‚â§ t‚ÇÇ.
    • A map f : (Œ±, t) ‚Üí (Œ≤, u) is continuous
      • iff t ‚â§ TopologicalSpace.induced f u (continuous_iff_le_induced)
      • iff TopologicalSpace.coinduced f t ‚â§ u (continuous_iff_coinduced_le).

    Topologies on Œ± form a complete lattice, with ‚ä• the discrete topology and ‚ä§ the indiscrete topology.

    For a function f : Œ± ‚Üí Œ≤, (TopologicalSpace.coinduced f, TopologicalSpace.induced f) is a Galois connection between topologies on Œ± and topologies on Œ≤.

    Implementation notes #

    There is a Galois insertion between topologies on Œ± (with the inclusion ordering) and all collections of sets in Œ±. The complete lattice structure on topologies on Œ± is defined as the reverse of the one obtained via this Galois insertion. More precisely, we use the corresponding Galois coinsertion between topologies on Œ± (with the reversed inclusion ordering) and collections of sets in Œ± (with the reversed inclusion ordering).
