    Construction of the Stone-ÄŒech compactification using ultrafilters.

    Implementation notes #

    Parts of the formalization are based on â€œUltrafilters and Topologyâ€ by Marius Stekelenburg, particularly section 5. However the construction in the general case is different because the equivalence relation on spaces of ultrafilters described by Stekelenburg causes issues with universes since it involves a condition on all compact Hausdorff spaces. We then define StoneCech Î± as T2Quotient (PreStoneCech Î±).
