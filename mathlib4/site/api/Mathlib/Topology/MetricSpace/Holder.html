    H√∂lder continuous functions #

    In this file we define H√∂lder continuity on a set and on the whole space. We also prove some basic properties of H√∂lder continuous functions.

    Main definitions #

    • HolderOnWith: f : X ‚Üí Y is said to be H√∂lder continuous with constant C : ‚Ñù‚â•0 and exponent r : ‚Ñù‚â•0 on a set s, if edist (f x) (f y) ‚â§ C * edist x y ^ r for all x y ‚àà s;
    • HolderWith: f : X ‚Üí Y is said to be H√∂lder continuous with constant C : ‚Ñù‚â•0 and exponent r : ‚Ñù‚â•0, if edist (f x) (f y) ‚â§ C * edist x y ^ r for all x y : X.

    Implementation notes #

    We use the type ‚Ñù‚â•0 (a.k.a. NNReal) for C because this type has coercion both to ‚Ñù and ‚Ñù‚â•0‚àû, so it can be easily used both in inequalities about dist and edist. We also use ‚Ñù‚â•0 for r to ensure that d ^ r is monotone in d. It might be a good idea to use ‚Ñù>0 for r but we don't have this type in mathlib (yet).
