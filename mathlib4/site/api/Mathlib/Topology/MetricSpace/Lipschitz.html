    Lipschitz continuous functions #

    A map f : Œ± ‚Üí Œ≤ between two (extended) metric spaces is called Lipschitz continuous with constant K ‚â• 0 if for all x, y we have edist (f x) (f y) ‚â§ K * edist x y. For a metric space, the latter inequality is equivalent to dist (f x) (f y) ‚â§ K * dist x y. There is also a version asserting this inequality only for x and y in some set s. Finally, f : Œ± ‚Üí Œ≤ is called locally Lipschitz continuous if each x : Œ± has a neighbourhood on which f is Lipschitz continuous (with some constant).

    In this file we specialize various facts about Lipschitz continuous maps to the case of (pseudo) metric spaces.
