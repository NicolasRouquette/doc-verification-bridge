    Congruences #

    This file defines Congruent, i.e., the equivalence between indexed families of points in a metric space where all corresponding pairwise distances are the same. The motivating example are triangles in the plane.

    Implementation notes #

    After considering two possible approaches to defining congruence ‚Äî either based on equal pairwise distances or the existence of an isometric equivalence ‚Äî we have opted for the broader concept of equal pairwise distances. This notion is commonly employed in the literature across various metric spaces that lack an isometric equivalence.

    For more details see the Zulip discussion.
