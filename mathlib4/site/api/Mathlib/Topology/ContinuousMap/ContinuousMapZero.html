    Continuous maps sending zero to zero #

    This is the type of continuous maps from X to R such that (0 : X) ‚Ü¶ (0 : R) for which we provide the scoped notation C(X, R)‚ÇÄ. We provide this as a dedicated type solely for the non-unital continuous functional calculus, as using various terms of type Ideal C(X, R) were overly burdensome on type class synthesis.

    Of course, one could generalize to maps between pointed topological spaces, but that goes beyond the purpose of this type.
