    Covering Maps #

    This file defines covering maps.

    Main definitions #

    • IsEvenlyCovered f x I: A point x is evenly covered by f : E ‚Üí X with fiber I if I is discrete and there is a homeomorphism f ‚Åª¬π' U ‚âÉ‚Çú U √ó I for some open set U containing x with f ‚Åª¬π' U open, such that the induced map f ‚Åª¬π' U ‚Üí U coincides with f.
    • IsCoveringMap f: A function f : E ‚Üí X is a covering map if every point x is evenly covered by f with fiber f ‚Åª¬π' {x}. The fibers f ‚Åª¬π' {x} must be discrete, but if X is not connected, then the fibers f ‚Åª¬π' {x} are not necessarily isomorphic. Also, f is not assumed to be surjective, so the fibers are even allowed to be empty.
