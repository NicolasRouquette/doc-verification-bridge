      The limit case of the induction #

      We relate linear independence in LocallyConstant (œÄ C (ord I ¬∑ < o')) ‚Ñ§ with linear independence in LocallyConstant C ‚Ñ§, where contained C o and o' < o.

      When o is a limit ordinal, we prove that the good products in LocallyConstant C ‚Ñ§ are linearly independent if and only if a certain directed union is linearly independent. Each term in this directed union is in bijection with the good products w.r.t. œÄ C (ord I ¬∑ < o') for an ordinal o' < o, and these are linearly independent by the inductive hypothesis.

      Main definitions #

      • GoodProducts.smaller is the image of good products coming from a smaller ordinal.

      • GoodProducts.range_equiv: The image of the GoodProducts in C is equivalent to the union of smaller C o' over all ordinals o' < o.

      Main results #

      • Products.limitOrdinal: for o a limit ordinal such that contained C o, a product l is good w.r.t. C iff it there exists an ordinal o' < o such that l is good w.r.t. œÄ C (ord I ¬∑ < o').

      • GoodProducts.linearIndependent_iff_union_smaller is the result mentioned above, that the good products are linearly independent iff a directed union is.
