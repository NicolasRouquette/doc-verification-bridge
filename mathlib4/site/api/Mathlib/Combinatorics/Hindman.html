    Hindman's theorem on finite sums #

    We prove Hindman's theorem on finite sums, using idempotent ultrafilters.

    Given an infinite sequence aâ‚€, aâ‚, aâ‚‚, â€¦ of positive integers, the set FS(aâ‚€, â€¦) is the set of positive integers that can be expressed as a finite sum of aáµ¢'s, without repetition. Hindman's theorem asserts that whenever the positive integers are finitely colored, there exists a sequence aâ‚€, aâ‚, aâ‚‚, â€¦ such that FS(aâ‚€, â€¦) is monochromatic. There is also a stronger version, saying that whenever a set of the form FS(aâ‚€, â€¦) is finitely colored, there exists a sequence bâ‚€, bâ‚, bâ‚‚, â€¦ such that FS(bâ‚€, â€¦) is monochromatic and contained in FS(aâ‚€, â€¦). We prove both these versions for a general semigroup M instead of â„•+ since it is no harder, although this special case implies the general case.

    The idea of the proof is to extend the addition (+) : M â†’ M â†’ M to addition (+) : Î²M â†’ Î²M â†’ Î²M on the space Î²M of ultrafilters on M. One can prove that if U is an idempotent ultrafilter, i.e. U + U = U, then any U-large subset of M contains some set FS(aâ‚€, â€¦) (see exists_FS_of_large). And with the help of a general topological argument one can show that any set of the form FS(aâ‚€, â€¦) is U-large according to some idempotent ultrafilter U (see exists_idempotent_ultrafilter_le_FS). This is enough to prove the theorem since in any finite partition of a U-large set, one of the parts is U-large.
