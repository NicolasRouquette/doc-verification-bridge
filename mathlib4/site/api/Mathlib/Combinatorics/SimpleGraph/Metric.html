    Graph metric #

    This module defines the SimpleGraph.edist function, which takes pairs of vertices to the length of the shortest walk between them, or ‚ä§ if they are disconnected. It also defines SimpleGraph.dist which is the ‚Ñï-valued version of SimpleGraph.edist.

    Main definitions #

    • SimpleGraph.edist is the graph extended metric.
    • SimpleGraph.dist is the graph metric.
