    Quivers #

    This module defines quivers. A quiver on a type V of vertices assigns to every pair a b : V of vertices a type a ‚ü∂ b of arrows from a to b. This is a generalization of Digraph V, which can be thought of as "a proposition a ‚ü∂ b of arrows".

    class Quiver (V : Type u) :
    Type (max u (v + 1))

    A quiver G on a type V of vertices assigns to every pair a b : V of vertices a type a ‚ü∂ b of arrows from a to b. This is hence a form of directed multigraphs.

    For graphs with no repeated edges, one can either use Quiver.IsThin to demand that the hom sets are subsingletons, or Digraph V (where the hom sets are Prop-valued).

    Because Category will later extend this class, we call the field Hom. Except when constructing instances, you should rarely see this, and use the ‚ü∂ notation instead.
