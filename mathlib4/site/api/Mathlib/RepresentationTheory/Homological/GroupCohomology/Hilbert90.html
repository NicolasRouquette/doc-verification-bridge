    Hilbert's Theorem 90 #

    Let L/K be a finite extension of fields. Then this file proves Noether's generalization of Hilbert's Theorem 90: that the 1st group cohomology $H^1(Aut_K(L), L^\times)$ is trivial. We state it both in terms of $H^1$ and in terms of cocycles being coboundaries.

    Hilbert's original statement was that if $L/K$ is Galois, and $Gal(L/K)$ is cyclic, generated by an element œÉ, then for every x : L such that $N_{L/K}(x) = 1,$ there exists y : L such that $x = y/œÉ(y).$ Using the fact that H¬π(G, A) ‚âÖ Ker(N_A)/(œÅ(g) - 1)(A) for any finite cyclic group G with generator g, we deduce the original statement from Noether's generalization.

    Noether's generalization also holds for infinite Galois extensions.
