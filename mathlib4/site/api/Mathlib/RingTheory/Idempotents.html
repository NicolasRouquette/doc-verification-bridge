    Idempotents in rings #

    The predicate IsIdempotentElem is defined for general monoids in Mathlib/Algebra/Group/Idempotent.lean; ring-specific lemmas are in Mathlib/Algebra/Ring/Idempotent.lean. In this file we provide various results regarding idempotent elements in rings.

    Main definitions #

    • OrthogonalIdempotents: A family { e·µ¢ } of idempotent elements is orthogonal if e·µ¢ * e‚±º = 0 for all i ‚â† j.
    • CompleteOrthogonalIdempotents: A family { e·µ¢ } of orthogonal idempotent elements is complete if ‚àë e·µ¢ = 1.

    Main results #

    • CompleteOrthogonalIdempotents.lift_of_isNilpotent_ker: If the kernel of f : R ‚Üí+* S consists of nilpotent elements, and { e·µ¢ } is a family of complete orthogonal idempotents in the range of f, then { e·µ¢ } is the image of some complete orthogonal idempotents in R.
    • existsUnique_isIdempotentElem_eq_of_ker_isNilpotent: If R is commutative and the kernel of f : R ‚Üí+* S consists of nilpotent elements, then every idempotent in the range of f lifts to a unique idempotent in R.
    • CompleteOrthogonalIdempotents.bijective_pi: If R is commutative, then a family { e·µ¢ } of complete orthogonal idempotent elements induces a ring isomorphism R ‚âÉ ‚àè R ‚ß∏ ‚ü®1 - e·µ¢‚ü©.
