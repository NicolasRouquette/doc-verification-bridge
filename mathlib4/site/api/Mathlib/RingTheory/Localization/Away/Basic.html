    Localizations away from an element #

    Main definitions #

    • IsLocalization.Away (x : R) S expresses that S is a localization away from x, as an abbreviation of IsLocalization (Submonoid.powers x) S.
    • exists_reduced_fraction' (hb : b ‚â† 0) produces a reduced fraction of the form b = a * x^n for some n : ‚Ñ§ and some a : R that is not divisible by x.
