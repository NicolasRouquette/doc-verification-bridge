    The Jacobi-Zariski exact sequence #

    Given R ‚Üí S ‚Üí T, the Jacobi-Zariski exact sequence is

    H¬π(L_{T/R}) ‚Üí H¬π(L_{T/S}) ‚Üí T ‚äó[S] Œ©[S/R] ‚Üí Œ©[T/R] ‚Üí Œ©[T/S] ‚Üí 0
