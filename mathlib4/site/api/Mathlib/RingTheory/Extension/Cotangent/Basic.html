    Naive cotangent complex associated to a presentation. #

    Given a presentation 0 ‚Üí I ‚Üí R[x‚ÇÅ,...,x‚Çô] ‚Üí S ‚Üí 0 (or equivalently a closed embedding S ‚Ü™ A‚Åø defined by I), we may define the (naive) cotangent complex I/I¬≤ ‚Üí ‚®Å·µ¢ S dx·µ¢ ‚Üí Œ©[S/R] ‚Üí 0.

    Main results #

    • Algebra.Extension.Cotangent: The conormal space I/I¬≤. (Defined in Generators/Basic)
    • Algebra.Extension.CotangentSpace: The cotangent space ‚®Å·µ¢ S dx·µ¢.
    • Algebra.Generators.cotangentSpaceBasis: The canonical basis on ‚®Å·µ¢ S dx·µ¢.
    • Algebra.Extension.CotangentComplex: The map I/I¬≤ ‚Üí ‚®Å·µ¢ S dx·µ¢.
    • Algebra.Extension.toKaehler: The projection ‚®Å·µ¢ S dx·µ¢ ‚Üí Œ©[S/R].
    • Algebra.Extension.toKaehler_surjective: The map ‚®Å·µ¢ S dx·µ¢ ‚Üí Œ©[S/R] is surjective.
    • Algebra.Extension.exact_cotangentComplex_toKaehler: I/I¬≤ ‚Üí ‚®Å·µ¢ S dx·µ¢ ‚Üí Œ©[S/R] is exact.
    • Algebra.Extension.Hom.Sub: If f and g are two maps between presentations, f - g induces a map ‚®Å·µ¢ S dx·µ¢ ‚Üí I/I¬≤ that makes f and g homotopic.
    • Algebra.Extension.H1Cotangent: The first homology of the (naive) cotangent complex of S over R, induced by a given presentation.
    • Algebra.H1Cotangent: H¬π(L_{S/R}), the first homology of the (naive) cotangent complex of S over R.

    Implementation detail #

    We actually develop these material for general extensions (i.e. surjection P ‚Üí S) so that we can apply them to infinitesimal smooth (or versal) extensions later.
