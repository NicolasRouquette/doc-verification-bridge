    The tensor product of R-algebras #

    This file provides results about the multiplicative structure on A ‚äó[R] B when R is a commutative (semi)ring and A and B are both R-algebras. On these tensor products, multiplication is characterized by (a‚ÇÅ ‚äó‚Çú b‚ÇÅ) * (a‚ÇÇ ‚äó‚Çú b‚ÇÇ) = (a‚ÇÅ * a‚ÇÇ) ‚äó‚Çú (b‚ÇÅ * b‚ÇÇ).

    Main declarations #

    • Algebra.TensorProduct.semiring: the ring structure on A ‚äó[R] B for two R-algebras A, B.
    • Algebra.TensorProduct.leftAlgebra: the S-algebra structure on A ‚äó[R] B, for when A is additionally an S algebra.

    • [C. Kassel, Quantum Groups (¬ßII.4)][Kassel1995]
