    Trace for (finite) ring extensions. #

    Suppose we have an R-algebra S with a finite basis. For each s : S, the trace of the linear map given by multiplying by s gives information about the roots of the minimal polynomial of s over R.

    Main definitions #

    • Algebra.embeddingsMatrix A C b : Matrix Œ∫ (B ‚Üí‚Çê[A] C) C is the matrix whose (i, œÉ) coefficient is œÉ (b i).
    • Algebra.embeddingsMatrixReindex A C b e : Matrix Œ∫ Œ∫ C is the matrix whose (i, j) coefficient is œÉ‚±º (b i), where œÉ‚±º : B ‚Üí‚Çê[A] C is the embedding corresponding to j : Œ∫ given by a bijection e : Œ∫ ‚âÉ (B ‚Üí‚Çê[A] C).
    • Module.Basis.traceDual: The dual basis of a basis under the trace form in a finite separable extension.

    Main results #

    • trace_eq_sum_embeddings: the trace of x : K(x) is the sum of all embeddings of x into an algebraically closed field
    • traceForm_nondegenerate: the trace form over a separable extension is a nondegenerate bilinear form
    • Module.Basis.traceDual_powerBasis_eq: The dual basis of a power basis {1, x, x¬≤...} under the trace form is a·µ¢ / f'(x), with f being the minpoly of x and f / (X - x) = ‚àë a·µ¢x‚Å±.
