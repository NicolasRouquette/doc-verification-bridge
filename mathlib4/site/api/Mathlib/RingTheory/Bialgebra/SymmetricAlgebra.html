    Bialgebra structure on SymmetricAlgebra R M #

    SymmetricAlgebra R M is the cocommutative commutative R-bialgebra on M in which each generator ι x is primitive: Δ(ι x) = ι x ⊗ 1 + 1 ⊗ ι x and ε(ι x) = 0.
