    Free rings #

    The theory of the free ring over a type.

    Main definitions #

    • FreeRing Œ± : the free (not commutative in general) ring over a type.
    • lift (f : Œ± ‚Üí R) : the ring hom FreeRing Œ± ‚Üí+* R induced by f.
    • map (f : Œ± ‚Üí Œ≤) : the ring hom FreeRing Œ± ‚Üí+* FreeRing Œ≤ induced by f.

    Implementation details #

    FreeRing Œ± is implemented as the free abelian group over the free monoid on Œ±.

    If Œ± is a type, then FreeRing Œ± is the free ring generated by Œ±. This is a ring equipped with a function FreeRing.of : Œ± ‚Üí FreeRing Œ± which has the following universal property: if R is any ring, and f : Œ± ‚Üí R is any function, then this function is the composite of FreeRing.of and a unique ring homomorphism FreeRing.lift f : FreeRing Œ± ‚Üí+* R.

    A typical element of FreeRing Œ± is a ‚Ñ§-linear combination of formal products of elements of Œ±. For example if x and y are terms of type Œ± then 3 * x * y * x - 2 * y * x + 1 is a "typical" element of FreeRing Œ±. In particular if Œ± is empty then FreeRing Œ± is isomorphic to ‚Ñ§, and if Œ± has one term t then FreeRing Œ± is isomorphic to the polynomial ring ‚Ñ§[t]. If Œ± has two or more terms then FreeRing Œ± is not commutative. One can think of FreeRing Œ± as the free non-commutative polynomial ring with coefficients in the integers and variables indexed by Œ±.
