    Dirichlet Characters #

    Let R be a commutative monoid with zero. A Dirichlet character œá of level n over R is a multiplicative character from ZMod n to R sending non-units to 0. We then obtain some properties of toUnitHom œá, the restriction of œá to a group homomorphism (ZMod n)À£ ‚Üí* RÀ£.

    Main definitions:

    • DirichletCharacter: The type representing a Dirichlet character.
    • changeLevel: Extend the Dirichlet character œá of level n to level m, where n divides m.
    • conductor: The conductor of a Dirichlet character.
    • IsPrimitive: If the level is equal to the conductor.
