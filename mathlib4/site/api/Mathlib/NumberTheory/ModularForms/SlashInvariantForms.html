    Slash invariant forms #

    This file defines functions that are invariant under a SlashAction which forms the basis for defining ModularForm and CuspForm. We prove several instances for such spaces, in particular that they form a module over ‚Ñù, and over ‚ÑÇ if the group is contained in SL(2, ‚Ñù).
