    Gaussian processes #

    This file contains basic properties of Gaussian processes. In particular, in IsGaussianProcess.of_isGaussianProcess, we show that if a stochastic process Y : S ‚Üí Œ© ‚Üí F is such that for each s : S, Y s can be written as a linear map applied to finitely many values of a certain Gaussian process, then Y is itself a Gaussian process.
