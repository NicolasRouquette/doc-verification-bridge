    Basic kernels #

    This file contains basic results about kernels in general and definitions of some particular kernels.

    Main definitions #

    • ProbabilityTheory.Kernel.deterministic (f : Œ± ‚Üí Œ≤) (hf : Measurable f): kernel a ‚Ü¶ Measure.dirac (f a).
    • ProbabilityTheory.Kernel.id: the identity kernel, deterministic kernel for the identity function.
    • ProbabilityTheory.Kernel.copy Œ±: the deterministic kernel that maps x : Œ± to the Dirac measure at (x, x) : Œ± √ó Œ±.
    • ProbabilityTheory.Kernel.discard Œ±: the Markov kernel to the type Unit.
    • ProbabilityTheory.Kernel.swap Œ± Œ≤: the deterministic kernel that maps (x, y) to the Dirac measure at (y, x).
    • ProbabilityTheory.Kernel.const Œ± (ŒºŒ≤ : measure Œ≤): constant kernel a ‚Ü¶ ŒºŒ≤.
    • ProbabilityTheory.Kernel.restrict Œ∫ (hs : MeasurableSet s): kernel for which the image of a : Œ± is (Œ∫ a).restrict s. Integral: ‚à´‚Åª b, f b ‚àÇ(Œ∫.restrict hs a) = ‚à´‚Åª b in s, f b ‚àÇ(Œ∫ a)
    • ProbabilityTheory.Kernel.comapRight: Kernel with value (Œ∫ a).comap f, for a measurable embedding f. That is, for a measurable set t : Set Œ≤, ProbabilityTheory.Kernel.comapRight Œ∫ hf a t = Œ∫ a (f '' t)
    • ProbabilityTheory.Kernel.piecewise (hs : MeasurableSet s) Œ∫ Œ∑: the kernel equal to Œ∫ on the measurable set s and to Œ∑ on its complement.
