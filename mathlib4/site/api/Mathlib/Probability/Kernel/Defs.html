    Markov Kernels #

    A kernel from a measurable space Œ± to another measurable space Œ≤ is a measurable map Œ± ‚Üí MeasureTheory.Measure Œ≤, where the measurable space instance on measure Œ≤ is the one defined in MeasureTheory.Measure.instMeasurableSpace.     Main definitions #

    Classes of kernels:

    • ProbabilityTheory.IsMarkovKernel Œ∫: a kernel from Œ± to Œ≤ is said to be a Markov kernel if for all a : Œ±, k a is a probability measure.
    • ProbabilityTheory.IsZeroOrMarkovKernel Œ∫: a kernel from Œ± to Œ≤ which is zero or a Markov kernel.
    • ProbabilityTheory.IsFiniteKernel Œ∫: a kernel from Œ± to Œ≤ is said to be finite if there exists C : ‚Ñù‚â•0‚àû such that C < ‚àû and for all a : Œ±, Œ∫ a univ ‚â§ C. This implies in particular that all measures in the image of Œ∫ are finite, but is stronger since it requires a uniform bound. This stronger condition is necessary to ensure that the composition of two finite kernels is finite.
    • ProbabilityTheory.IsSFiniteKernel Œ∫: a kernel is called s-finite if it is a countable sum of finite kernels.
