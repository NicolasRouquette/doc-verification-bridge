    Monad Operations for Probability Mass Functions #

    This file constructs two operations on PMF that give it a monad structure. pure a is the distribution where a single value a has probability 1. bind pa pb : PMF Î² is the distribution given by sampling a : Î± from pa : PMF Î±, and then sampling from pb a : PMF Î² to get a final result b : Î².

    bindOnSupport generalizes bind to allow binding to a partial function, so that the second argument only needs to be defined on the support of the first argument.
