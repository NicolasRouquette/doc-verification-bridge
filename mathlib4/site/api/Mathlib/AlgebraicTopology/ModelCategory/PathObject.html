    Path objects #

    We introduce a notion of path object for an object A : C in a model category. It consists of an object P, a weak equivalence Œπ : A ‚ü∂ P equipped with two retractions p‚ÇÄ and p‚ÇÅ. This notion shall be important in the definition of "right homotopies" in model categories.

    This file dualizes the definitions in the file Mathlib/AlgebraicTopology/ModelCategory/Cylinder.lean.

    Implementation notes #

    The most important definition in this file is PathObject A. This structure extends another structure PrepathObject A (which does not assume that C has a notion of weak equivalences, which can be interesting in situations where we have not yet obtained the model category axioms).

    The good properties of path objects are stated as typeclasses PathObject.IsGood and PathObject.IsVeryGood.

    The existence of very good path objects in model categories is stated in the lemma PathObject.exists_very_good.

    References #

    • [Daniel G. Quillen, Homotopical algebra][Quillen1967]
    • https://ncatlab.org/nlab/show/path+space+object

    A pre-path object for A : C is the data of a morphism Œπ : A ‚ü∂ P equipped with two retractions.

    • P : C

      the underlying object of a (pre)path object

    • p‚ÇÄ : self.P ‚ü∂ A

      the first "projection" from the (pre)path object

    • p‚ÇÅ : self.P ‚ü∂ A

      the second "projection" from the (pre)path object

    • Œπ : A ‚ü∂ self.P

      the diagonal of the (pre)path object

    • Œπ_p‚ÇÄ : CategoryTheory.CategoryStruct.comp self.Œπ self.p‚ÇÄ = CategoryTheory.CategoryStruct.id A
    • Œπ_p‚ÇÅ : CategoryTheory.CategoryStruct.comp self.Œπ self.p‚ÇÅ = CategoryTheory.CategoryStruct.id A
