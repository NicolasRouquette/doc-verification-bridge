    Perfect Secrecy #

    Characterisation theorems for perfect secrecy following [KatzLindell2020], Chapter 2.

    Main results #

    • Cslib.Crypto.Protocols.PerfectSecrecy.EncScheme.perfectlySecret_iff_ciphertextIndist: ciphertext indistinguishability characterization ([KatzLindell2020], Lemma 2.5)
    • Cslib.Crypto.Protocols.PerfectSecrecy.EncScheme.perfectlySecret_keySpace_ge: Shannon's theorem, |K| ≥ |M| ([KatzLindell2020], Theorem 2.12)

    References #

    • [J. Katz, Y. Lindell, Introduction to Modern Cryptography][KatzLindell2020]
