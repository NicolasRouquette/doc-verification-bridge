    Default Artifact Paths for RL Examples #

    TorchLean's runnable RL examples write small JSON artifacts to data/rl/ by default:

    • training curves (TrainLog JSON),
    • optional policy snapshots, and
    • optional episode path snapshots.

    Both the executable trainers (under NN/Examples/Models/RL) and the editor-side viewers (under NN/Examples/RL) need to agree on these defaults. Keeping the strings here gives both sides one shared path convention.

    Users can always override these paths with the corresponding CLI flags (e.g. --log, --policy, --path).
