    Session Shape and Index Operations #

    This file contains the session-level operations that preserve or rearrange tensor shape: activation helpers, reshapes, indexing, gathers, broadcasts, and reductions. Each operation dispatches through the same eager/compiled session boundary as the lower-level tensor ops.
