    Batch Lifting #

    batchPointwise n model wraps a single-example model σ → τ into a batched model (dim n σ) → (dim n τ) by running the underlying model once per batch element.

    Correctness-first batch lift for exposing PyTorch-like N×... APIs even when a primitive only exists for the unbatched shape.
