    Learning-rate schedulers (public API) #

    This module contains the pure schedule math for learning-rate policies used by TorchLean training loops and examples. The core entrypoint is Config together with lrAt.

    References #

    • PyTorch schedulers: torch.optim.lr_scheduler.* (https://pytorch.org/docs/stable/optim.html#how-to-adjust-learning-rate)

    Type

    Small learning-rate scheduler surface for higher-level training code.

    This file keeps the interface compact: a Config is just a description of a schedule, and lrAt cfg t computes the learning rate at step/epoch index t.

    PyTorch mapping #

    Config.step and Config.exponential correspond to the schedule math of:

    • torch.optim.lr_scheduler.StepLR
    • torch.optim.lr_scheduler.ExponentialLR
    • constant (lr : Float) : Config
    • step (base : Float) (stepSize : ℕ) (gamma : Float := 0.1) : Config
    • exponential (base gamma : Float) : Config
