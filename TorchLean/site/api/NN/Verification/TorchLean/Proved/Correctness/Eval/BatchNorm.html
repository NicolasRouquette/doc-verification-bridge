    BatchNorm IR Evaluation #

    The TorchLean compiler, PyTorch importer, verifier, and PyTorch exporter all meet at the IR BatchNorm node. This file records the small semantic fact that matters at that boundary: once the payload is present and the input has matching NCHW channels, IR evaluation is the standard eval-mode BatchNorm formula applied independently at each (N,C,H,W) coordinate.
