    Geometry3D CLI wrapper #

    This module keeps the runnable camera-box3d-cert entrypoint next to the reusable verifier.

    The example artifacts are generated by scripts/verification/geometry3d/* under _external. The default path here is a deterministic sample certificate so the CLI can be exercised without network access, GPU access, or model downloads.
