    Forward-error lemmas for the rounded NF backend.

    This module collects the public NF operation facts used by runtime-approximation proofs: scalar rounding bounds, tensor-level arithmetic and activation bounds, rounded reductions, and the FwdNode constructors that package those facts for graph composition.
